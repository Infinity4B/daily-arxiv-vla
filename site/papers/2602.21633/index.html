Self-Correcting VLA: Online Action Refinement via Sparse World Imagination

2026-02-25 · 原文链接 · 幻觉翻译 · 2602.21633

研究单位

  • 作者未明确列出所属机构，但推测为 AI/机器人学习研究团队（根据论文内容和 GitHub 项目 Kisaragi0/SC-VLA）。

论文概述

  • 提出 Self-Correcting VLA (SC-VLA) 框架，通过稀疏世界想象 (Sparse World Imagination, SPI) 和在线动作优化 (Online Action Refinement, OAR)，实现视觉-语言-动作模型的自我改进 (self-improvement)。
  • 旨在解决传统 VLA 模型依赖静态数据先验、缺乏物理动态理解，以及传统强化学习依赖外部奖励信号而与模型内部状态脱节的问题。

核心贡献

  • 提出 SC-VLA 框架，将离线动作生成与在线优化相结合，通过稀疏世界想象 (SPI) 预测未来状态，约束策略编码物理演化。
  • 设计了在线动作优化 (OAR) 模块，利用残差强化学习和基于想象的内生密集奖励 (endogenous dense rewards) 动态调整轨迹方向。
  • 在模拟（ManiSkill3 基准）和真实世界（ARX5 机器人平台）的复杂操作任务上进行了系统性评估，证明了该方法在成功率和任务吞吐量上的优越性。

方法描述

  • 方法基于条件流匹配 (Conditional Flow Matching, FM) 和 Soft Actor-Critic (SAC) 算法构建。
  • 第一阶段 (SPI)：在 VLM-DiT 骨干网络上，添加辅助预测头，同时预测任务进度 (progress) 和未来相对状态变化 (ΔState)，作为稀疏世界信号。
  • 第二阶段 (OAR)：冻结基础策略，在其输出之上训练一个轻量级残差策略 (Residual Policy)。奖励信号由环境稀疏奖励和基于预测未来状态方向的内生密集指导奖励 (intrinsic dense guidance reward) 动态加权组合而成。

数据集与资源

  • 使用 ManiSkill3 仿真基准中的四个挑战性机器人操作任务（StackCube, PlaceSphere, LiftPegUpright, PegInsertion）。
  • 在真实世界使用 ARX5 机械臂平台进行验证。
  • 模型基于 SigLIP-2 视觉编码器、Eagle-2 VLM 和 DiT (Diffusion Transformer) 骨干网络。
  • 训练资源未在提供的片段中明确说明。

评估与结果

  • 评估环境：ManiSkill3 仿真环境与 ARX5 真实机器人。
  • 主要指标：任务成功率，成功任务的平均完成步数（任务吞吐量）。
  • 关键结果：

- 在 ManiSkill3 上，SC-VLA 以 86% 的平均成功率超越所有基线（如 GR00T N1.5 72%， π₀ 55%），并在最难的 PegInsertion 任务上提升 28%。

- 在成功任务中，SC-VLA 的平均步数（157步）比最佳基线减少了 16%，实现了最高吞吐量。

- 在真实世界任务中，SC-VLA 获得 71% 的平均成功率，比最佳基线 GR00T N1.5 高出 14%。

- 消融实验验证了稀疏世界想象、内生密集奖励和动态权重调度各组件均对性能有显著贡献。
