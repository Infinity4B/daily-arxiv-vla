Card 03 核心贡献

核心贡献

  • 提出4D-VLA模型，整合空间模块生成3D感知的视觉token，缓解坐标系和状态混乱。
  • 设计记忆库采样方法，高效利用历史信息。
  • 在LIBERO和真实世界任务中显著超越基线，如在LIBERO-LONG任务上成功率比OpenVLA高25.4%。
  • 构建MV-Bench多视角数据集，验证模型的空间理解和新视角泛化能力。
Card 04 方法描述

方法描述

  • 输入处理：使用序列RGB-D图像，通过3D坐标嵌入生成空间感知视觉token。
  • 时空编码：将历史帧与当前帧融合，加入可学习时间位置编码，形成4D表示。
  • 记忆库采样：自适应选择信息量大的历史帧，避免均匀采样的冗余。
  • 模型架构：基于InternVL-4B，冻结视觉编码器，训练其他参数。
  • 损失函数：结合平移、旋转、夹爪和方向损失，其中方向损失强调动作方向准确性。
Card 05 数据集与资源

数据集与资源

  • 数据集：DROID（真实世界，76k轨迹）、LIBERO（仿真，多任务集）、自建MV-Bench（多视角）。
  • 训练资源：8块NVIDIA A6000 GPU，训练约96小时，预训练 batch size 512，微调batch size 128。
