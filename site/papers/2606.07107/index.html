Card 02 论文概述

论文概述

  • 提出 Coarse-to-Control 框架，将规划（planning）引入视觉-语言-动作（VLA）模型的动作 token 空间，而非依赖文本或图像中间表示
  • 解决直接动作生成在长时程任务中因缺乏明确规划而易导致早期错误累积的问题
  • 通过先预测粗粒度动作 token 规划未来轨迹，再基于该规划生成可执行动作 token，实现更精准的推理与控制
Card 03 核心贡献

核心贡献

  • 指出精准电机控制受益于中间运动级意图，而动作空间比文本或图像更适合作为该意图的表示媒介
  • 提出 Coarse-to-Control 规划-执行 VLA 架构，利用联合 tokenizer 将粗规划 token 与可执行动作 token 置于共享离散词汇表中，使规划更贴近控制流形
  • 在 LIBERO、SimplerEnv-WidowX 和真实世界操作任务上验证了动作 token 规划始终优于直接动作生成，并超越文本、视觉和空间链式思考基线
Card 04 方法描述

方法描述

  • 设计 双粒度动作联合 tokenizer：执行模式（m=0）对短时程动作块进行 tokenize，规划模式（m=1）对压缩后的长时程粗轨迹进行 tokenize，二者共享动作词汇表
  • 训练 规划-执行自回归 VLA：策略先自回归生成粗规划 token，再以该规划为条件生成可执行 token，推理时仅解码可执行 token 为机器人动作，规划作为隐式上下文
  • 粗动作获取：通过将未来动作序列分块求和（运动）和取最后状态（夹爪）得到压缩的粗轨迹，表示阶段级运动意图
Card 05 数据集与资源

数据集与资源

  • 数据集：LIBERO（Spatial、Object、Goal、Long 四套件）、SimplerEnv-WidowX（四个 WidowX 操作任务）、自建真实世界操作数据集（每任务 50 条演示）
  • 模型规模：文中未明确说明参数量，但基于 VLA 架构，采用联合动作 tokenizer 和自回归策略
  • 训练资源：未明确说明，但提及使用快速训练变体（如 Faster-AR），可能使用多 GPU 训练
Card 06 评估与结果

评估与结果

  • 评估环境：LIBERO 模拟（每任务 50 次 rollout）、SimplerEnv-WidowX 模拟（每任务 24 次 rollout）、真实世界四任务（每任务 20 次 rollout）
  • 主要指标：任务成功率（%）
  • 关键结果：

- LIBERO 平均成功率 97.90%，其中 Long 套件达 95.0%，显著优于所有基线

- SimplerEnv-WidowX 平均成功率 83.3%，在 Put Spoon、Put Carrot、Stack Block 上均达最优

- 真实世界多阶段任务平均成功率 62.5%，在长时程任务上优势明显，规划使执行步骤更高效（减少无效往复）

- 消融实验表明：共享规划-执行 token 空间 + 足够长的规划时域（160 步）能带来最大增益
