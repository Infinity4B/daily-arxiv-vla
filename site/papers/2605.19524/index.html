SafeAlign-VLA: A Negative-Enhanced Safe Alignment Framework for Risk-Aware Autonomous Driving

2026-05-19 · 原文 · 翻译 · 2605.19524

提出了 SafeAlign-VLA 框架，这是一个面向风险感知自动驾驶的负样本增强安全对齐框架。 解决现有 VLA 模型依赖正向专家演示、缺乏对负样本（高风险行为）学习的问题。 目标是使模型不仅学习模仿专家行为，还能明确识别并远离高风险轨迹，从而提升决策的安全性和鲁棒性。

Card 02 论文概述

论文概述

  • 提出了 SafeAlign-VLA 框架，这是一个面向风险感知自动驾驶的负样本增强安全对齐框架。
  • 解决现有 VLA 模型依赖正向专家演示、缺乏对负样本（高风险行为）学习的问题。
  • 目标是使模型不仅学习模仿专家行为，还能明确识别并远离高风险轨迹，从而提升决策的安全性和鲁棒性。
Card 03 核心贡献

核心贡献

  • 提出了一个两阶段安全对齐框架，在监督微调（SFT）和强化学习（RL）中均利用负样本，实现故障分析、轨迹修正和风险感知决策。
  • 设计了 反事实安全配对（CSP） 数据构建范式，能从高风险场景中提取结构化监督信息并生成正轨迹替代方案。
  • 引入了 锚点基于 GRPO 方法，利用正负轨迹作为对比锚点，通过群组相对优势优化策略，引导采样并惩罚高风险行为。
  • 在 NAVSIM 和 DeepAccident 基准上进行了全面实验验证，证明了方法的有效性。
Card 04 方法描述

方法描述

  • 采用 Vision-Language-Action（VLA） 规划架构，将视觉输入与语言提示融合，通过 LLM 主干生成推理文本和动作轨迹。
  • 提出 反事实安全配对（CSP） 数据构建范式，通过计算最小碰撞时间（TTC）识别高风险样本，并通过纵向重规划生成反事实安全轨迹，形成正负样本对。
  • 采用两阶段训练策略：第一阶段为渐进式课程 SFT，使模型具备基础场景理解与轨迹修正能力；第二阶段为锚点基于 GRPO 强化学习，冻结 LLM 主干，仅微调注意力池化层和动作头。
  • 设计了包含轨迹质量、偏好安全性和目标导向性的加权奖励函数，用于强化学习阶段的策略优化。
Card 05 数据集与资源

数据集与资源

  • 使用 NAVSIM 数据集（约103k训练样本，12k测试样本）和 DeepAccident 数据集（2496样本，含948碰撞案例）。
  • 基础模型采用 Qwen2.5-VL，并应用 LoRA 进行参数高效微调。
  • 所有实验在 4块 NVIDIA A6000 GPU 上完成。
Card 06 评估与结果

评估与结果

  • 评估基准：NAVSIM v1 测试集和 DeepAccident 测试集。
  • 主要评估指标：PDMS（规划质量分数）、碰撞率、语言准确率、风险预测准确率。
  • 关键结果：在 NAVSIM v1 测试集上达到 89.1 PDMS，相比无负样本基线提升 1.3%；在 DeepAccident 上，碰撞率降至 3.36%，同时实现 84.2% 语言准确率和 85.8% 风险预测准确率。
