Card 02 论文概述

论文概述

  • 提出 HAT-4D，首个从单目视频中重建多物体 4D 交互（几何、动态、物理交互）的智能体框架
  • 解决单目 4D 重建中因严重遮挡、深度歧义和复杂动态导致现有方法失效的问题
  • 通过引入人机协作（Human-in-the-loop）和交互知识图谱（IKG），生成物理合理且时序一致的 4D 内容
Card 03 核心贡献

核心贡献

  • 提出 HAT-4D 框架，集成多智能体系统与多层级人机协作，从单目视频重建多物体 4D 交互
  • 引入 交互知识图谱 (IKG) 作为因果推理引擎，显式编码长期物理变化，指导 3D 生成和 4D 传播
  • 构建 MVOIK-4D 基准数据集，包含 112 个场景、77 个任务、39 种交互类别，并提出多维度评估协议
Card 04 方法描述

方法描述

  • 使用 VLM 从视频中提取 交互知识图谱 (IKG)，将视频划分为事件段，并编码物体实体、交互语义、深度排序和空间关系
  • 基于 IKG 调度 3D 物体生成与组合技能，利用 SAM3D 重建物体并优化 6DoF 位姿，构成静态场景
  • 采用 记忆增强的 4D 传播算子 (L4GM)，将 3D 场景渲染成正交平面图，结合记忆库进行时序传播
  • 引入多层级人机协作，通过 高斯级、区域级、物体级 细化算子，以及在线微调，将人类知识注入生成循环
Card 05 数据集与资源

数据集与资源

  • 构建 MVOIK-4D 数据集，包含 ToolOIK-4D（工具交互）和 MemoryOIK-4D（时空记忆）两个子集
  • 使用 Qwen3-VL 235B 作为基础 VLM，SAM3D 和 L4GM 分别作为 3D/4D 生成算子
  • 训练资源：使用 lr=1×10⁻⁵ 进行位姿优化，记忆库大小设为 8
Card 06 评估与结果

评估与结果

  • 评估基准：MVOIK-4D 数据集，对比基线包括 L4GM、GVFDiffusion、SV4D、STAG4D、FB4D
  • 评估指标：生成质量 (CLIP, LPIPS, FVD)、交互质量 (Deform, Relation)、记忆质量 (Intra, Long)
  • 关键结果：HAT-4D 在 LPIPS、FVD、交互质量和记忆质量上均取得最优，少量人工干预即可显著提升重建性能
