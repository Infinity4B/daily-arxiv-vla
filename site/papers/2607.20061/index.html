ReferTrack: Referring Then Tracking for Embodied Visual Tracking

Card 02 论文概述

论文概述

  • 提出 ReferTrack，一种面向具身视觉跟踪的“先指认后跟踪”范式，使用单一前向相机
  • 解决具身视觉跟踪中目标识别与轨迹规划两个核心挑战，特别是将目标识别转化为基于图像空间的索引边界框选择问题
  • 通过构建 Refer-CoT 令牌和滑动窗口边界框队列，实现从自然语言指令到图像落地推理再到运动规划的端到端统一
Card 03 核心贡献

核心贡献

  • 提出 Referring-Then-Tracking 范式：将目标识别转化为从索引边界框目录中选择一个条目的受限多选问题，取代抽象空间潜变量推理
  • 设计 TVBI (Temporal-Viewpoint-Bbox Indicator) 令牌：通过滑动窗口队列注入目标几何特征到视觉历史中，保留目标运动线索
  • 构建 Refer-QA 数据集：基于 SYNTH-PEDES 数据集生成，与 EVT 导航数据协同训练，强化视觉落地能力
  • 在 EVT-Bench 上实现单视图最优性能，单相机结果超越多个多相机基线，且无需强化学习
  • 在 Unitree Go2 和 Unitree G1 机器人上实现真实世界部署，验证模拟到真实的迁移能力
Card 04 方法描述

方法描述

  • 双流架构：视觉流使用 SigLIP+DINOv2 双编码器提取特征，通过 TVBI 令牌注入历史边界框几何信息；候选目录使用 YOLO11+ByteTrack 检测行人并构建索引边界框集合
  • Refer-CoT 决策：LLM 在第一阶段从候选目录中选择一个索引令牌作为指代输出，第二阶段以此为基础预测轨迹航点
  • 滑动窗口队列：维护历史边界框队列，用于未来帧的 TVBI 编码，训练时注入噪声以模拟跟踪错误，推理时自回归更新
  • 训练策略：两阶段 SFT，第一阶段对齐视觉投影器，第二阶段联合训练导航数据和 Refer-QA 数据（1:1 比例），使用三种损失函数（轨迹回归、指代分类、文本生成）
Card 05 数据集与资源

数据集与资源

  • EVT-Bench 训练集：1.3M 专家跟踪轨迹，来自 Habitat 3.0 模拟器
  • SYNTH-PEDES 数据集：用于生成 1.3M Refer-QA 样本
  • 模型规模：4B 参数，基于 Qwen3-4B LLM 主干
  • 检测器：YOLO11 + ByteTrack
  • 训练资源：仅使用监督微调，无强化学习
Card 06 评估与结果

评估与结果

  • 评估基准：EVT-Bench（Habitat 3.0 模拟器），单前向视图设置
  • 评估指标：成功率 (SR)、跟踪率 (TR)、碰撞率 (CR)
  • 关键结果：

- 单目标跟踪 (STT)：89.4% / 92.5% / 1.6

- 干扰跟踪 (DT)：73.3% / 81.8% / 7.6

- 歧义跟踪 (AT)：74.1% / 85.7% / 7.7

- 超越 TrackVLA++ (DT SR 提升 6.8%，AT SR 提升 22.9%)

- 超越 VLingNav (DT 性能) 等多个单视图方法

- 匹配甚至超越多个多相机基线方法

- 真实机器人部署：平均 10.6 Hz 推理频率，检测仅需 12 ms
