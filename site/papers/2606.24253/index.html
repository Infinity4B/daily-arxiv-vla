Card 02 论文概述

论文概述

  • 针对当前视觉Transformer（ViT）训练需要海量标注数据、高分辨率/动态分辨率下计算成本过高、固定分辨率预训练与下游多模态大模型（VLM）需求不匹配的核心痛点，提出TuringViT，旨在降低SOTA级ViT的训练、定制与部署门槛，让高性能视觉编码器对更广泛的社区可及
  • 核心设计包含三个关键模块：Turing Linear Attention (TLA) 线性注意力架构、VISTA-Curation 图文视频数据策展pipeline、原生动态分辨率预训练范式，从架构、数据、训练三个维度降低SOTA ViT的获取成本
  • TuringViT仅用10%的训练数据量即可超越主流开源ViT基线，同时已作为统一视觉基座落地于小鹏内部的VLA 2.0、智能座舱、Iron Robotics等多类AI系统，为社区提供了可复现的低成本SOTA ViT训练方案
Card 03 核心贡献

核心贡献

  • 提出Turing Linear Attention (TLA) 作为主导注意力算子，配合稀疏插入的标准softmax注意力层，将高分辨率输入下的计算复杂度从二次方降至近线性，大幅改善了推理延迟随分辨率增长的 scaling 曲线，让高分辨率训练部署成为可能
  • 设计VISTA-Curation 多模态数据策展pipeline，通过图文相对评分、视频时序聚合等策略提升训练样本的监督质量，实现数据效率的显著提升，仅用10%数据即可超越基线模型
  • 实现全流程原生动态分辨率预训练，从预训练阶段就支持可变分辨率输入，无需后处理适配，天然匹配下游VLM的输入需求，避免了固定分辨率预训练后适配的性能损失
  • 提供可复现的SOTA ViT训练完整pipeline，验证了模型在零样本分类、密集预测、下游VLM等多类任务上的性能，且scaling实验显示模型仍远未达到饱和
  • 模型已作为统一视觉基座落地于小鹏汽车VLA 2.0、智能座舱、Iron Robotics等实际业务场景，验证了其在实际工业场景中的可用性
Card 04 方法描述

方法描述

  • 架构设计：核心为Turing Linear Attention (TLA) 线性注意力，采用SiLU作为核函数，引入输入相关的输出门和序列长度感知归一化，稳定可变长度token序列的训练，同时保留局部和高频视觉响应；每5层TLA稀疏插入1层标准MHA层，维持显式token间交互；整体采用RMSNorm、SwiGLU前馈网络，搭配2D RoPE支持原生分辨率输入，无需位置嵌入插值
  • 数据策展（VISTA-Curation）：图文部分先过滤低质量图像，生成多候选描述后通过相对视觉语言评分+文本信息熵评分联合排序，筛选出视觉对齐度高、语义丰富的描述；视频部分通过时序分解、均匀帧采样、语义-运动一致性过滤、局部-全局描述融合验证，将原始长视频转化为高质量的时序监督样本，提升训练的时序建模能力
  • 预训练范式：采用四阶段渐进式预训练，依次为MIM特征初始化、范围约束动态分辨率图文预训练、原生分辨率图文精调、混合图文视频对齐，逐步适配下游VLM的输入模式，兼顾训练效率和最终性能
Card 05 数据集与资源

数据集与资源

  • 训练数据集：基于DataComp-1B、LAION-2B等公开数据集，经VISTA-Curation策展得到约8.5亿图文对，外加200万自策展的视频文本对
  • 模型规模：两个公开变体，TuringViT-18L（共18层，15层TLA+3层MHA）、TuringViT-24L（共24层，20层TLA+4层MHA），patch大小为16×16，共享相同的嵌入维度、注意力头维度和MLP比例
  • 训练资源：总训练消耗约53452张H800 GPU小时，分四个阶段分别为9984、30720、12288、460 H800 GPU小时
Card 06 评估与结果

评估与结果

  • 评估环境与基准：零样本分类涵盖ImageNet-1K、ImageNet-v2、ObjectNet、ImageNet-A、ImageNet-R、ImageNet-Sketch共6个基准；检索涵盖MS-COCO、Flickr30K；密集预测涵盖NYUv2深度估计、ADE20K语义分割、DAVIS-2017零样本视频追踪；下游VLM任务涵盖RefCOCO、CountBench、BLINK、OCRBench、ChartQA、TextVQA等
  • 主要评估指标：分类任务用准确率，检索任务用召回率，深度估计用RMSE（越低越好），语义分割用mIoU（越高越好），追踪任务用$\mathcal{J}\&\mathcal{F}$分数（越高越好），VLM任务用对应任务的准确率/得分
  • 关键实验结果：

- 零样本分类平均得分83.6，检索平均得分78.9，TuringViT-24L仅用0.85B数据即可超越使用10B数据的SigLIP2-L（分类平均83.4），数据效率提升约10倍

- 密集预测任务中，TuringViT-18L在1024分辨率下深度估计RMSE为53.7、语义分割mIoU为43.2、零样本追踪$\mathcal{J}\&\mathcal{F}$为47.5；TuringViT-24L对应指标为50.7、45.0、48.5，优于同量级基线

- 下游VLM任务中，TuringViT-24L在RefCOCO平均得分90.4、CountBench得分81.1、OCRBench V1得分798，均优于SigLIP2-L和MobileCLIP2-L

- 消融实验显示动态分辨率训练、数据策展、数据过滤等模块均带来显著性能提升，其中动态分辨率训练的单步增益最大；数据scaling实验显示模型在850M数据规模下仍未出现饱和，仍有进一步提升空间

- 推理延迟方面，TuringViT在高分辨率输入下的延迟 scaling 曲线远低于标准softmax注意力ViT，在RTX 3080Ti上测试的1K分辨率吞吐量优势明显，硬件部署效率更高
