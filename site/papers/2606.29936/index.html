OpenSPM: An Environment-Transferable Robotic Key Spatial Pose Memory and Closed-Loop High-Frequency Flow-Matching Action Generation Model

2026-06-29 · 原文 · 翻译 · 2606.29936

针对开放环境桌面机器人操作中语义理解、精确几何位姿估计和高频动作生成的要求，以及现有VLA模型缺乏显式几何约束且训练成本高的问题，提出了OpenSPM框架。 框架结合空间位姿记忆和流匹配动作生成，利用语义条件3D感知与卡尔曼滤波跟踪连续6D位姿，从人类演示中提取关键空间位姿作为可转移的记忆。 在推理时检索相关记忆条目，通过SE(3)变换将位姿转移到新场景，并通过轻量级流匹配模型生成高频动作块，结合闭环反馈抑制轨迹误差累积。

Card 03 核心贡献

核心贡献

  • 语义条件3D感知：提出了统一多视角2D语义、3D重建和卡尔曼状态估计的感知管线，提供连续且尺度一致的6D位姿表示。
  • 关键空间位姿记忆库：设计了基于检索的记忆库，将长序列演示压缩为阶段标签、以对象为中心的SE(3)相对位姿，映射语言指令到可执行几何先验。
  • 闭环流匹配生成：引入轻量级（240K参数）条件流匹配模型生成关键位姿间的高频动作块，集成实时状态反馈和终端残差校正防止误差累积。
  • 系统评估与高效性：在LIBERO-GOAL上实现85.6%的成功率和1033.3Hz的等效控制频率，且仅需极低的推理计算力，并通过消融实验验证了结构化记忆和闭环校正的必要性。
Card 04 方法描述

方法描述

  • 3D感知与卡尔曼估计：利用多视角2D分割（SAM等）和3D重建获取观测，通过卡尔曼滤波预测与更新步骤平滑视觉抖动，输出连续6D位姿。
  • 记忆库构建与检索：离线提取夹爪状态变化等关键边界位姿，存储为末端执行器相对于锚点对象的相对位姿；在线根据语义和几何一致性检索最佳匹配条目。
  • 相对位姿转移：通过新场景中对象的估计位姿与存储的相对位姿相乘，将演示中的关键位姿转换到新场景，保持局部交互几何不变。
  • 条件流匹配动作生成：输入当前位姿、目标位姿、相对误差及进度作为条件，使用ODE积分沿学习到的速度场生成长度为5的动作块。
  • 闭环执行与终端残差校正：每个动作块以实时夹爪位姿为条件；在关键抓取/放置节点，若残差超阈值则迭代生成校正动作直至误差收敛。
Card 05 数据集与资源

数据集与资源

  • LIBERO-GOAL数据集（10个语言条件桌面操作任务，共500个评估回合）
  • 动作生成模型规模：240K参数的轻量级MLP
  • 推理延迟：4.8 ms（等效频率1033.3Hz），极低的推理AI算力需求
Card 06 评估与结果

评估与结果

  • 评估基准：LIBERO-GOAL十任务基准，包含Diffusion Policy、OpenVLA、WorldVLA等基线对比
  • 主要指标：任务成功率（SR）、动作块生成延迟、等效动作输出频率、平均执行步数
  • 关键结果：85.6%成功率与1033.3 Hz等效频率，优于其他参数量远大于此的基线模型（如OpenVLA 79.2% / 4.9Hz，Octo-Base 84.6% / 54.7Hz）
  • 卡尔曼位姿预测平均平移MAE为5.8mm，平均旋转MAE为2.12度
  • 消融实验：无记忆库的零样本多模态推理成功率降至23.8%；开环流匹配成功率仅3.0%，加入闭环提升至29.8%，加入终端残差校正恢复至85.6%
