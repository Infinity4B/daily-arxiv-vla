Card 02 论文概述

论文概述

  • 提出Geometric Action Model (GAM)，一种语言条件驱动的机器人操作策略模型
  • 核心目标是将预训练的几何基础模型（GFM） 直接转化为机器人策略，使其能同时进行3D场景理解、未来状态预测和动作生成
  • 解决现有视觉-语言-动作模型（VLA）和世界-动作模型（WAM）主要依赖2D图像先验、缺乏显式3D几何推理能力的问题
Card 03 核心贡献

核心贡献

  • 提出GAM架构，通过在GFM的中间层插入因果未来预测器，将GFM改造为共享的感知、预测和动作解码基座
  • 实现动作与几何在统一令牌空间中的联合预测，单个自回归序列和单次前向传播即可同时生成动作和未来深度图
  • 在多个仿真和真实机器人操作基准测试中，证明GAM在准确率、鲁棒性、推理速度和模型参数量方面均优于现有基于基础模型的方法
  • 特别在相机视角扰动等需要几何理解的场景下，GAM展现出显著的性能优势
Card 04 方法描述

方法描述

  • 拆分GFM：在预训练GFM的中间层（第12层）将其分为浅层观察编码器和深层解码器
  • 插入因果未来预测器：在拆分处加入12层因果Transformer，以语言指令、本体感知和动作历史为条件，预测未来的几何特征令牌和动作令牌
  • 特征传播与解码：将预测的未来令牌送入剩余GFM深层，同时解码未来深度图和机器人动作块
  • 训练目标：联合优化动作回归损失、未来特征对齐损失和未来深度重建损失，实现端到端训练
Card 05 数据集与资源

数据集与资源

  • 预训练数据集：RoboCasa365（78.4万轨迹）、MimicGen 和 OpenX-Embodiment，合计784K单臂机器人轨迹
  • 模型骨干：基于 DA3-Giant 几何基础模型，总参数量约 1.4B
  • 训练资源：使用AdamW优化器，冻结浅层编码器和深度头，在仿真基准上后训练
Card 06 评估与结果

评估与结果

  • 评估基准：LIBERO（标准操作）、LIBERO-Plus（鲁棒性测试，含相机、光照、背景等7种扰动）以及真实机器人操作任务
  • 主要指标：任务成功率
  • 关键结果：

- 在LIBERO-Plus上，GAM成功率最高（85.5%），尤其在相机扰动场景下领先第二名9.7个百分点

- 真实世界实验中，GAM在标准场景和相机扰动场景下均显著优于基线模型

- 推理速度达到6.9ms/步，比可比基线（如Cosmos-Policy 382.4ms）快55倍，参数仅1.4B
