Card 03 核心贡献

核心贡献

  • 提出了 ConsisVLA-4D，一个在 3D 感知和 4D 推理中推进时空一致性的高效创新框架
  • 设计了 CV-Aligner 和 CO-Fuser，分别确保跨视图对象语义一致性和跨对象空间几何一致性，提升模型对复杂 3D 场景的理解
  • 开发了 CS-Thinker 和时空一致性注意力机制（SC-Attn），增强 4D 时空推理，确保模型在动态环境中生成一致且可靠的动作
  • 在 LIBERO 基准和真实机器人平台上实现了 21.6% 和 41.5% 的性能提升，以及 2.3× 和 2.4× 的推理加速
Card 04 方法描述

方法描述

  • CV-Aligner：通过显式语义对象选择（Explicit Semantic Object Selection）过滤冗余视觉表示，仅保留与指令相关的对象，然后使用单融合（Single-Fusion）跨视图对齐对象身份，生成高效的空间表示
  • CO-Fuser：通过组融合（Group-Fusion）处理多视图以解决单视图对象位置的几何模糊，使用隐式几何关系聚合（Implicit Geometric Relation Aggregation）细化对象间的空间关系
  • CS-Thinker：学习两套隐式知识——使用 CV-Aligner 的语义 token 预测未来固定视图中的动态对象，使用 CO-Fuser 的几何 latent token 推断跨所有视图的全局深度，在推理时这些知识占观察-指令序列的不到 10%

数据集与资源

  • 训练配置：支持单臂任务和双臂任务的训练

  • 计算效率：推理速度比 OpenVLA 快 2.3×（模拟）和 2.4×（真实世界）
  • 视觉输入压缩：仅使用约 1/8-1/12 的原始视觉 token，显著降低了计算开销
