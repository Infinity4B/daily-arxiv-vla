Card 02 论文概述

论文概述

  • 提出 Lift3D-VLA，一个统一的视觉-语言-动作（VLA）框架，旨在赋予机器人显式3D点云推理和时序一致的动作生成能力
  • 针对现有VLA模型缺乏几何理解、3D数据有限以及无法联合建模3D几何与动态动作序列的问题，通过继承并改进 Lift3D 策略，将2D基础模型提升至3D感知，并引入自监督几何预训练和逐层时序动作建模
  • 在大规模机器人数据上预训练，在22个仿真任务和8个真实世界任务中取得最优性能，并展现出对分布外扰动的强泛化能力
Card 03 核心贡献

核心贡献

  • 提出 Lift3D-VLA 框架，将显式3D点云推理与VLA模型结合，支持大规模机器人预训练，扩展了先前工作 Lift3D，增加双臂任务评估
  • 设计 Geometry-Centric Masked Autoencoding (GC-MAE)，一种双目标自监督学习方案，同时重建当前点云并预测未来几何演变，使VLA视觉编码器学习鲁棒的3D物理表示
  • 开发 逐层时序动作建模（Layer-wise Temporal Action Modeling），利用LLM中间层到深层持续预测动作块，提升动态环境中动作的时序一致性
  • 通过合成大规模3D点云数据（基于VGGT），并利用自监督训练增强视觉编码器，避免了对昂贵3D标注的依赖
Card 04 方法描述

方法描述

  • 架构：基于 Prismatic VLM 骨干，使用 SigLIP 和 DINOv2 双视觉编码器，共享编码器处理2D图像和3D点云；LLM采用 7B LLaMA2，通过投影层融合多模态令牌
  • 2D模型提升策略：将3D点云投影到多个虚拟平面，对齐预训练2D位置嵌入，通过平均聚合得到3D位置嵌入，实现点云在2D视觉编码器中的直接编码，最小化几何信息损失
  • GC-MAE：利用 VGGT 从RGB生成伪点云作为监督信号，对可见的点令牌进行掩码，通过轻量级Transformer解码器重建被掩码点的坐标，并预测下一帧可见点的几何位置，使用 Chamfer Distance 作为损失，通过 LoRA 高效微调视觉编码器
  • 逐层动作建模：根据动作块长度，从LLM的深层均匀选取多个层，每层负责预测一个未来动作步，使用 DDPM 扩散模型进行动作生成，推理时采用 DDIM 加速，实现时序连贯的动作序列输出
Card 05 数据集与资源

数据集与资源

  • 预训练数据：整合 Open-X-Embodiment、DROID、RoboMIND 等开源机器人数据集，构建400K轨迹（28M帧）用于机器人基础预训练；利用VGGT从140K轨迹合成3D点云，用于GC-MAE自监督训练
  • 模型规模：LLM部分为 7B LLaMA2，视觉编码器基于SigLIP和DINOv2
  • 训练资源：GC-MAE训练15个epoch，batch size 4096，学习率1e-4，使用AdamW优化器及余弦学习率调度；未明确提及GPU配置
Card 06 评估与结果

评估与结果

  • 仿真环境：在 MetaWorld 和 RLBench 共22个任务上进行单任务和多任务评估，与 CLIP、R3M、VC-1、PointNet、DP3、Lift3D 等基线对比
  • 真实世界：在8个操作任务（含单臂和双臂）上评估，与最强真实世界基线比较
  • 主要指标：任务成功率（Success Rate, S.R.）
  • 关键结果：

- 在MetaWorld和RLBench上，Lift3D-VLA 的平均成功率分别比先前最佳VLA方法高出 10.8% 和 11.1%

- 在真实世界任务中，Lift3D-VLA 的成功率比最强基线高 4个百分点

- 消融实验验证了 GC-MAE 和逐层时序动作建模对性能提升的显著贡献，同时在未见物体、光照和背景变化下展现出强泛化能力
