Card 02 论文概述

论文概述

  • 提出 GEAR-VLA 框架，通过学习统一的几何感知动作表示来解决现有 Vision-Language-Action (VLA) 模型在真实世界部署中的泛化问题
  • 针对未见过物体、背景偏移和不同机器人 embodiment 的挑战，解决现有 VLA 缺乏统一几何感知表示、依赖低层轨迹监督、3D 特征与语义空间不对齐以及 embodiment 差异导致的泛化能力差等问题
  • 在仿真基准（LIBERO、LIBERO-Plus、RoboTwin 2.0）、真实世界双臂操作（AgileX、LDT-01）和大规模通用抓取任务上验证强泛化能力
Card 03 核心贡献

核心贡献

  • 提出 GEAR-VLA 框架，结合粗到细动作学习（coarse-to-fine action learning）与语义对齐 3D 集成（semantic-aligned 3D integration），学习统一的几何感知动作表示
  • 设计 Embodiment Canonicalization 机制，通过相对末端执行器动作空间（Relative End-effector Action）和轻量级状态投影器，使策略无需机器人特定语义提示即可跨 embodiment 迁移
  • 在 LIBERO、LIBERO-Plus、RoboTwin 2.0 上达到 state-of-the-art 性能，并在真实世界双臂操作和包含 212 个未见物体的大规模抓取基准上验证强泛化能力
Card 04 方法描述

方法描述

  • 粗到细策略学习：先进行 embodied VLM 预训练，使用 FAST 风格动作 token 和因果 VQ-VAE 生成的潜在动作 ID 提供离散监督，学习具身推理和动作语义；随后通过梯度解耦的 DiT（Diffusion Transformer）动作专家生成连续动作，使用 flow matching loss 训练，并通过 stop-gradient 防止连续动作优化破坏 VLM 主干网络
  • 语义对齐 3D 几何集成：在冻结的原始 2D 视觉通路旁添加可训练的 VGGT 3D 空间编码器，通过零初始化的扩展视觉投影器（expanded visual projector）将 2D 语义特征与 3D 结构特征拼接后映射到 LLM 视觉 token 空间，实现几何感知与语义表示的融合
  • Embodiment 规范化：采用相对末端执行器动作（SE(3) 相对变换）替代绝对动作或步进 delta 动作，减少对机器人基座坐标的依赖并避免动作块内的误差累积；状态输入采用末端执行器姿态与关节角度的配对表示，通过轻量级 embodiment 特定状态投影器映射到共享表示空间，实现两阶段轻量级跨机器人适配
Card 05 数据集与资源

数据集与资源

  • 仿真数据集：LIBERO、LIBERO-Plus（含 7 种扰动设置的零样本泛化）、RoboTwin 2.0（含强 domain randomization 的大规模双臂操作基准）
  • 真实世界数据集：AgileX（14-DoF 双臂机器人，3 个任务，每任务 200 条示范）、LDT-01（16-DoF 双臂机器人，训练中未见过的 embodiment）
  • 通用抓取基准：212 个未见过的物体，覆盖轴对称、块状、不规则和工具类物体，3 种场景设置（稀疏/密集 clutter、背景/光照变化），共 6,360 次真实机器人试验
  • 模型基础：基于 Qwen2.5-VL 初始化 2D 视觉编码器和 LLM，使用 VGGT 作为 3D 空间主干；动作序列以 30 Hz 重采样，30 步动作块（1 秒预测 horizon）
  • 训练资源：粗到细预训练使用 240 个 H200 GPU（每 GPU batch size 8 训练 350K iterations，batch size 4 训练 700K iterations）；下游微调使用 56 个 H200 GPU（batch size 4，12K iterations）
Card 06 评估与结果

评估与结果

  • 评估环境：标准仿真基准（LIBERO、LIBERO-Plus、RoboTwin 2.0）、真实世界双臂操作平台（AgileX、LDT-01）、大规模通用抓取基准
  • 主要指标：任务成功率（success rate）
  • 关键实验结果：

- 在 LIBERO 上达到 98.7% 平均成功率，超越所有先前方法

- 在 LIBERO-Plus 零样本迁移中达到 88.7% 成功率，超越最强基线 π₀.₅（85.7%）2.1 个点

- 在 RoboTwin 2.0 上达到 Clean 91.1% 和 Randomized 89.9% 成功率，超越 ACoT 11.00 和 11.20 个点

- 在真实世界 AgileX 上达到 85.9% 平均成功率，在未见过的 LDT-01 embodiment 上达到 81.0%，超越 π₀.₅ 7.1 个点

- 在大规模通用抓取基准上达到 90.1% 成功率，超越 π₀.₅（79.1%）和 DexGraspVLA（84.4%），且仅需首帧目标 mask 即可完成抓取
