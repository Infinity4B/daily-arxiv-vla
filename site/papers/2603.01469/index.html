Mean-Flow based One-Step Vision-Language-Action

2026-03-02 · 原文链接 · 幻觉翻译 · 2603.01469

研究单位

  • 论文作者未明确列出所属机构（从HTML标题和作者信息推断为独立研究者）

论文概述

  • 提出了一种名为 Mean-Flow based One-Step VLA 的新型视觉-语言-动作模型，旨在显著提升机器人操作任务中的动作生成效率
  • 解决了现有基于FlowMatching的VLA模型因迭代采样需求导致的生成延迟问题，通过引入“平均流”概念，实现单步或极少步的快速高质量动作生成

核心贡献

  • 首次将MeanFlow生成方法集成到VLA框架中，提出Mean-Flow based One-Step VLA架构
  • 用平均去噪向量场预测器替代传统的FlowMatching机制，大幅提升生成效率，速度比当前最优方法快8.7倍
  • 在真实机器人实验上验证了框架，证明其在单步和多步生成模式下均能实现鲁棒性能，且不需要预训练、蒸馏或额外的复杂性启发式方法

方法描述

  • 核心技术是MeanFlow方法：该方法学习一个时间区间内的平均去噪向量场，而非瞬时向量场，从而直接建模从噪声端点到数据端点的整体映射
  • 引入了“MeanFlow Identity”作为核心数学原理，连接瞬时向量场与平均向量场，并使用自适应损失（Adaptive Loss）代替标准L2损失以提高训练稳定性
  • 动作生成策略：在推理时，条件于视觉语言模型特征，MeanFlow动作专家直接预测平均向量场，并通过A_0 = A_1 - u_θ(A_1, 0, 1)公式，一步从噪声生成目标动作序列

数据集与资源

  • 使用了三个专门收集的真实机器人操作任务数据集（PickPlace、Stacking、Sorting），每个任务包含100条演示轨迹，共300条轨迹
  • 模型架构：使用SmolVLM-2作为冻结的视觉语言模型骨干，搭配基于Transformer的Mean-Flow动作专家
  • 训练资源：在NVIDIA RTX 4090 GPU上进行训练。在线推理和数据收集在配备Intel Core i5-10200H和NVIDIA GTX 1650 GPU的笔记本电脑上执行

评估与结果

  • 评估环境：在自制的SO-101六自由度机械臂上，使用多视角RGB相机进行真实机器人实验
  • 主要评估指标：任务成功率和动作生成速度
  • 关键实验结果：所提One-Step VLA在单步生成（NFE=1）下，动作生成速度分别是SmolVLA和Diffusion Policy的8.7倍和83.9倍。在三个任务上的平均成功率为78%，与SmolVLA（84.5%）和Diffusion Policy（61.5%）相比具有竞争力
