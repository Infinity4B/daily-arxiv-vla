Benchmarking Robot Memory Under Interference

2026-06-21 · 原文 · 翻译 · 2606.22338

Card 03 核心贡献

核心贡献

  • 提出首个面向机器人的跨会话干扰记忆基准RoboMME-Interference，通过在查询任务前插入k∈{0,1,3,7}个来自不同任务族的无关会话，可控量化记忆系统的长上下文抗干扰能力
  • 公开完整的评估结果网格，覆盖9种记忆系统、9个任务族的全部测试表现，同步公开代码、数据集、可视化分析与项目资源（项目主页：https://robotmemorybench.com）
  • 实证证明当前记忆增强VLA普遍缺乏长上下文抗干扰记忆能力：感知类记忆系统在无干扰时可显著提升任务成功率，但随无关会话积累性能快速衰减，最终接近无记忆 baseline 水平
Card 04 方法描述

方法描述

  • 基准构建逻辑：基于公开机器人记忆基准RoboMME扩展，选取其16个任务中具备独立先验演示段的9个任务族，每个查询轮次关联一个外部历史缓冲区：缓冲区先存放该查询对应的相关先验演示，再插入k个来自不同任务族的无关会话作为干扰，每个无关会话为32帧的固定长度单元，保证不同干扰条件下的可比性
  • 评估系统范围：选取RoboMME发布的8种记忆增强π₀.₅变体，搭配无记忆的π₀.₅ baseline共9个系统，覆盖感知类（FrameSamp、TokenDrop）、循环类（TTT）三类记忆表示，以及Context、Modulator、Expert三种记忆整合机制
  • 评估流程：所有系统无需针对本基准重新训练，直接通过未修改的原推理机制消费缓冲区内容，在仿真环境下完成任务成功率测试
  • 创新点与关键技术：

- 首次将机器人记忆评估维度从单轮次扩展到跨会话长上下文场景，引入可控无关会话干扰，可量化记忆系统的抗干扰能力

- 采用跨任务族的无关会话作为干扰源，避免同类任务干扰带来的矛盾信息，保证干扰强度的可解释性与评估公平性

- 全流程不修改原有记忆系统的推理逻辑，保证评估结果的公平可比

Card 05 数据集与资源

数据集与资源

  • 基础数据集：基于RoboMME基准构建，选取9个具备独立先验演示的任务族，包括4个模仿任务（MoveCube、InsertPeg、PatternLock、RouteStick）、3个视频条件参考任务（VideoRepick、VideoPlaceButton、VideoPlaceOrder）、2个视频条件持久性任务（VideoUnmask、VideoUnmaskSwap），每个任务族包含50个测试轮次
  • 评估模型：共9个系统，包括无记忆π₀.₅ baseline，以及8种RoboMME公开的记忆增强变体，无额外新增的大规模模型，均使用公开预训练checkpoint
  • 训练资源：无额外训练消耗，所有评估直接使用RoboMME发布的预训练checkpoint，通过系统原有推理机制运行，未进行针对性的重训练或微调
Card 06 评估与结果

评估与结果

  • 评估环境：基于RoboMME仿真环境，每个系统在每个任务族、每个k值（0,1,3,7）条件下完成全部测试轮次，结果报告Wilson 95%置信区间
  • 主要评估指标：任务成功率（准确率）
  • 关键实验结果：

- 无干扰（k=0）时，感知类记忆系统可显著提升性能，其中FrameSamp-Modul表现最优，相比无历史baseline（17.3%）提升27.1个百分点，达到45.3%的成功率

- 随无关会话数量增加，所有系统性能均出现明显衰减：FrameSamp-Modul从k=0的45.3%下降到k=7的19.3%，衰减约26个百分点，最终接近无历史baseline水平；TokenDrop-Modul从35.3%下降到19.8%，衰减约16个百分点

- 记忆整合机制对性能影响显著：相同感知表示下，Modulator整合机制的表现远优于Context和Expert机制；循环类TTT记忆系统在所有历史长度下均无法获得有效性能提升，始终接近无历史baseline水平

- 任务属性影响记忆增益：简单和中等难度任务中记忆系统的增益更明显，困难任务中所有系统均接近成功率地板；不同任务族对记忆的敏感度存在显著差异
