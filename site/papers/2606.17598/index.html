MuseVLA: An Adaptive Multimodal Sensing Vision-Language-Action Model for Robotic Manipulation

2026-06-16 · 原文 · 翻译 · 2606.17598

Card 03 核心贡献

核心贡献

  • 提出 MuseVLA 模型，实现了灵巧手操作中传感器作为按需工具的可扩展和高效集成。
  • 引入 sensor tokens 实现自适应传感器选择，并提出 grounded sensor image 作为异构传感器数据的统一中间表示。
  • 开发 数据合成管线，从现有 RGB 数据集生成多模态训练数据，实现对未见任务的零样本泛化。
  • 在具有挑战性的多模态感知机器人操作任务上验证了模型的优越性能，显著超越基线方法。
Card 04 方法描述

方法描述

  • 模型架构采用 PaliGemma-2 作为 VLM 主干，DiT 作为动作专家，利用 VITRA 权重进行初始化。
  • VLM 主干生成特定的 sensor tokens（如 、、）及目标描述，类似带有参数的工具调用。
  • 使用 SAM3 分割模型基于目标描述生成掩码，将传感器热力图叠加在 RGB 图像的掩码区域上，构建 grounded sensor image。
  • 训练采用 VLM 损失（传感器令牌与目标描述生成）与扩散模型 MSE 损失的联合优化，实现端到端协同训练。
Card 05 数据集与资源

数据集与资源

  • 基于现有 RGB 数据集 MolmoAct、AgiBotWorld-Alpha 和 VITRA 合成多模态数据集（9.6K episodes，1.05M 帧）。
  • 自采集真实世界多模态数据集：720 个远程操作演示片段，覆盖 10 种指令和 7 种物体。
  • 训练硬件：64 张 A100 (40GB) GPU，Batch size 512，训练 20K 步（约 20 小时）。
Card 06 评估与结果

评估与结果

  • 评估环境：配备 Robotera XHand 灵巧手及多传感器（热成像、毫米波雷达、麦克风阵列）的真实机器人平台。
  • 主要指标：Task success rate（任务成功率）和 Task score（包含感知与操作阶段的得分）。
  • 在包含预训练的情况下，平均任务成功率达到 80.6%，显著优于纯 RGB 基线（20.8%）和原始传感器热力图基线（33.3%）。
  • 在未见任务上展现出强大的零样本泛化能力，平均成功率达到 66.7%。
  • 自适应传感器选择将推理 GPU 内存消耗从 13.23 GB 降低至 6.61 GB。
