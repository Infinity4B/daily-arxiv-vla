Card 02 论文概述

论文概述

  • 本文提出Lumo-2，一种面向机器人控制的潜在世界-动作模型，通过潜在空间中的世界动态推理实现预测性动作生成，替代传统显式推理机制，在降低推理 token 复杂度的同时捕获丰富的物理动态和动作相关依赖。
  • 针对现有视觉-语言-动作（VLA）和世界-动作模型（WAM）中动作模态与视觉、语言、潜在世界动态对齐不足、重建型动作分词存在重建精度与下游控制性能不匹配的问题，本文提出三阶段渐进式模态预对齐训练范式，逐步提升动作表征的语义结构化程度。
  • 论文同时提出上下文区分机制和块级自回归解码策略，分别解决单步观测的部分可观测歧义和实时控制的推理效率问题，最终实现可泛化、可扩展的具身智能。
Card 03 核心贡献

核心贡献

  • 提出Lumo-2潜在世界-动作模型架构，以潜在世界动态作为隐式推理桥梁，将动作生成 conditioned 在物理 grounded 的潜在动态表征上，实现预测性推理，替代传统显式推理机制，降低 token 复杂度的同时提升动态建模能力。
  • 提出三阶段渐进式模态预对齐训练范式，逐步将动作表征与潜在世界动态、视觉、语言对齐，解决重建型动作分词的重建精度与下游控制性能 misalignment 问题，显著提升表征的语义结构化程度和模型缩放性能。
  • 提出上下文区分短期记忆机制，通过历史动作队列编码缓解单步观测导致的动作歧义问题，同时保证训练和推理效率；提出块级自回归（BAR）解码策略，将逐 token 自回归改为块级并行生成，实现2.71倍的推理加速，满足实时机器人控制需求。
  • 通过系统的实证研究，验证了潜在世界建模和模态对齐对缩放定律与分布外泛化的核心作用，证明结构化多模态对齐结合预测性推理是推进可泛化具身智能的有效路径。
Card 04 方法描述

方法描述

  • Lumo-2基于预训练的Qwen3.5-4B视觉-语言模型构建，核心是将动作生成过程与潜在世界动态建模耦合：模型首先从多视角视觉观测、语言指令和历史动作中推断潜在世界动态表征，再基于该表征生成未来动作序列，实现预测性推理。
  • Stage1采用双分支向量量化（VQ）架构联合训练视觉自编码分支和动作自编码分支，通过双向约束（潜在世界动态引导动作重建、动作表征正则化潜在空间）实现动作与潜在世界动态的早期双模态对齐，得到物理 grounded 的潜在动态空间和带全局上下文感知的动作编码器。
  • Stage2引入语义模块和多任务学习框架，将动作表征升级为语义丰富的形式，通过动作重建、行为理解、视觉-语言引导动作生成、跨模态预测、跨模态对比判别等多任务，实现视觉-语言-动作的统一模态对齐，同时保留Stage1预训练的物理动态建模能力。
  • Stage3进行大规模协同训练，融合VLM数据、视频数据和机器人数据，在保留通用多模态理解能力的同时强化具身推理能力；此外引入上下文区分机制和BAR解码策略，分别解决感知歧义和推理效率问题。
  • 关键技术创新包括：用潜在推理替代显式推理降低复杂度、渐进式模态预对齐解决跨模态对齐难题、BAR机制实现高效并行解码、上下文区分机制缓解部分可观测性挑战。
Card 05 数据集与资源

数据集与资源

  • 机器人数据集：包含自采的Astribot S1操作数据、跨具身机器人数据集（覆盖AGIBot Genie-1、Galaxea R1 Pro、RealSource RS-02、ARX、Agile X、YAM等多款机器人平台）、第一人称人类视频数据集（EgoTaskQA、EPIC-KITCHENS、Ego4D）。
  • VLM数据集：curated 的约5300万样本大规模多模态语料，覆盖通用理解、定位、规划、认知四大类能力，包含FineVision、FineVideo、ADE20K、COCOStuff、Molmo2等公开数据集来源。
  • 模型规模：基座模型为Qwen3.5-4B，扩展词汇表包含32个视觉上下文token、1024个语义动作token、1个动作序列占位token，动作分支将动作分解为躯干、双臂的平移、旋转、夹爪运动共8个语义组独立编码。
  • 训练资源：Stage1训练使用64张NVIDIA H100 GPU，单GPU batch size为24，采用bfloat16混合精度训练；Stage2使用64张H100 GPU；Stage3使用160张H100 GPU，采用序列打包（最大长度8192 token）和动态分辨率提升训练效率。
Card 06 评估与结果

评估与结果

  • 评估环境与基准：在真实世界机器人平台上开展实验，评估基准涵盖具身理解类（BLINK-Depth、BLINK-Spatial、CV-Bench、Ego-Plan2等）、具身定位类（RefSpatialBench、ShareRobot、Where2place等）、通用拾取放置、长时间跨度操作、灵巧操作等任务，所有VLM评估通过lmms-eval框架开展。
  • 主要评估指标：VLM任务采用准确率，拾取放置任务采用指令遵循率（IFR）和任务成功率（SR），额外评估动作重建误差、动作语义预测准确率、推理延迟等指标。
  • 关键实验结果：

- 具身感知能力上，Lumo-2在多项具身基准上超越Qwen3.5 4B、RoboBrain-2.5 4B、Rynnbrain 4B等4B级SOTA模型，Stage3协同训练未出现严重灾难性遗忘， retains 纯视觉语言能力。

- 潜在世界动态验证：学习到的潜在表征能捕捉动作相关的未来状态变化，跨具身（不同机器人、人类）的同类动作在潜在空间中存在语义对齐；仅用潜在动态预测任务指令的准确率达90%，接近5帧观测的94% baseline，显著高于单帧观测的43%。

- 泛化能力上，在通用拾取放置任务的未见指令、未见物体设置中，Lumo-2的IFR和SR均显著超越π0.5和Fast-WAM基线，在需要推理和泛化的场景中提升尤为明显。

- 效率与缩放上，渐进式模态预对齐使动作表征的语义判别性显著提升，缩放曲线优于重建型基线；块级自回归（BAR）策略将端到端推理延迟降低至93.53ms，相比标准自回归提升2.71倍，满足实时控制需求。

- 真实任务性能上，在需要时序推理、物理理解、长时间跨度及灵巧操作的挑战性真实任务中，Lumo-2一致优于VLA和WAM类基线，且支持使用第一人称人类视频、VisionPro数据等易获取的数据进行有效微调。
