Card 03 核心贡献

核心贡献

  • 提出SC3-Eval自洽视频生成方案，通过正向-逆向动力学一致性、跨视角一致性、测试时一致性三类互补的一致性目标，将预训练视频基础模型适配为准确的闭环策略评估器
  • 提出无需修改训练流程的不确定性驱动提前终止准则，复用逆向动力学模式输出的每动作块一致性误差作为可靠性信号，在rollout漂移时提前终止，无需额外监督即可提升闭环可靠性
  • 在7个真实世界视觉-语言-动作（VLA）策略上的实验表明，SC3-Eval达到最优策略评估性能，可泛化到分布外的任务语义，且能复现单轨迹级别的策略失败模式而非仅预测聚合成功率
Card 04 方法描述

方法描述

  • 基于统一动力学模型架构（如UVA），以Cosmos3-Nano的预训练权重为初始化，构建支持正向动力学（从动作预测未来帧）、逆向动力学（从帧恢复动作）、跨视角修复三类任务的统一模型，所有模式共享同一套Transformer参数
  • 训练阶段联合优化三个一致性目标：正向-逆向动力学一致性通过共享参数联合训练正向和逆向模式，锚定生成的rollout到物理可行的动作流形，抑制自回归漂移；跨视角一致性通过训练模型从剩余视角修复被遮挡的相机视角，保证多视角观测的长期 coherence，无需额外记忆机制
  • 推理阶段采用测试时一致性：通过正向-逆向动力学的一致性误差作为每动作块的不确定性信号，超过阈值时提前终止rollout；同时采用预测-执行horizon解耦策略，训练时使用更长的预测horizon保留预训练先验、提升动态拟合效果，推理时仅使用前l帧作为下一轮观察上下文
Card 05 数据集与资源

数据集与资源

  • 训练数据集：381小时真实世界桌面清理（table bussing）数据集，包含12类物体（餐具、垃圾等），采集自单一物理场景，包含两个第三人称相机、一个手腕相机的三视角同步流，原生分辨率480×640，采样率20Hz，同时额外下采样至10Hz形成多帧率训练混合；分布外评估使用reverse table bussing任务，仅交换物体-目标映射，未出现在训练集中
  • 模型规模：基于Cosmos3-Nano预训练视频基础模型构建的统一动力学模型，未额外披露具体参数量

Card 06 评估与结果

评估与结果

  • 评估环境与基准：在真实世界桌面清理场景中评估，覆盖7个基于π0.5架构的VLA策略检查点，任务分为分布内（table bussing）和分布外（reverse table bussing）两类，评估模式包括离线开环（用真实动作序列驱动世界模型，隔离视频生成质量）和在线闭环（策略作用于生成的帧，模拟真实交互）
  • 主要评估指标：Pearson相关系数r（衡量预测成功率与真实世界成功率的线性相关性，反映评估器的绝对校准度）、MMRV（Mean Maximum Rank Violation）（衡量策略排序的一致性，越低说明策略排名和真实世界越一致，是实际checkpoint选择中最相关的指标）
  • 关键实验结果：SC3-Eval在闭环场景下整体Pearson r=0.929，MMRV=0.119，超过Ctrl-World、IRASim、Cosmos-Predict 2.5三个强基线；分布内任务在线评估Pearson r达0.984，超过离线模式的0.959；在所有四个（离线/在线×分布内/分布外）设置中均取得最优MMRV；可泛化到分布外的reverse table bussing任务；在四类失败模式（无失败、语言遵循失败、物体抓取失败、物体放置失败）的复现任务中均取得最高的复现率，远超基线
