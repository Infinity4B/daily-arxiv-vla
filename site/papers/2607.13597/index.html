Card 02 论文概述

论文概述

  • 核心问题：Vision-Language-Action（VLA）模型仅通过动作监督微调时，会逐步侵蚀从预训练视觉语言模型中继承的意图级语义结构，导致分布外泛化能力下降，且缺乏对“高质量动作表示”的量化衡量标准
  • 研究目标：通过诊断性分析揭示动作表示语义结构的侵蚀规律，提出无需修改推理部署结构的即插即用方法修复该结构，提升VLA的泛化性能
  • 关键发现：动作-指令对齐质量与分布外任务成功率、单轨迹 rollout 结果高度同步，可作为表示质量的代理指标
Card 03 核心贡献

核心贡献

  • 提出动作表示的语义结构是衡量表示质量的可量化指标，系统性验证发现仅动作微调会侵蚀该结构，且其质量与分布外泛化、单轨迹任务成功率正相关
  • 提出即插即用的语义锚定方法，通过对比学习将动作表示锚定到语义流形，结合共享-私有分解解决语义对齐与动作预测的梯度冲突，所有辅助模块推理时丢弃，部署模型与基线完全一致
  • 在不同架构的VLA骨干、仿真与真实机器人基准上验证方法有效性，真实场景内部分布任务成功率最高提升+18.7%，分布外泛化最高提升+21.5%
Card 04 方法描述

方法描述

  • 诊断阶段：提取VLA第10层动作token隐藏状态，经注意力池化后，用双向InfoNCE损失与冻结预训练文本编码器的指令嵌入对齐，跟踪微调全程的对齐度变化
  • 训练阶段：将每个动作token特征分解为共享通道（编码意图级语义）和私有通道（保留 Embodiment 细节、时序等执行专属信息），仅共享通道经注意力池化后与指令嵌入做对比对齐，同时引入重建损失保证特征可重建、去相关损失避免两通道重新纠缠
  • 创新点：受镜像神经元理论启发，用共享-私有分解解决语义对齐（要求表面不变性）和动作预测（要求物理细节敏感性）的目标冲突，所有辅助模块仅训练时使用，零推理开销
Card 05 数据集与资源

数据集与资源

  • 仿真数据集：LIBERO（4个任务套件，每任务50条演示，评估近分布性能）、LIBERO-Pro（LIBERO扩展扰动基准）、SimplerEnv（基于BridgeData V2多源真实机器人数据集，评估WidowX操作任务的实到 Sim 域迁移性能）
  • 真实机器人平台：AgileX Cobot Magic V2双臂6-DoF机器人，配置平行夹爪和3个RGB相机，4个任务家族（水果抓取放置、碗碟堆叠、纸箱包装、橱柜收纳），5个分布外评估轴（语言、位置、物体、视觉、任务组合）
  • 模型规模：π₀（PaliGemma-2B视觉语言骨干+Gemma-300M动作专家，全参数微调）、SpatialVLA（4B参数自动回归VLA，LoRA秩32微调）
  • 训练资源：统一使用AdamW优化器、bfloat16混合精度、梯度裁剪范数1.0；π₀在LIBERO上训练30k步（batch size 32，峰值学习率2.5×10⁻⁵），BridgeData V2上训练50k步（batch size 128，峰值学习率5×10⁻⁵），真实机器人任务训练30k步（batch size 64，峰值学习率5×10⁻⁵）；SpatialVLA采用DeepSpeed ZeRO Stage-1，约22k步（2轮epoch），峰值学习率10⁻⁴
Card 06 评估与结果

评估与结果

  • 评估基准：仿真场景覆盖LIBERO近分布、LIBERO-Pro扰动、SimplerEnv跨域迁移；真实场景覆盖4个任务家族的内部分布与5个维度的分布外泛化
  • 核心评估指标：任务成功率（内部分布/分布外）、动作-指令对齐Recall@1、单轨迹余弦相似度
  • 关键实验结果：

- 诊断实验：π₀微调过程中动作-指令对齐从预训练阶段62.74%先骤降、后部分恢复、再持续下降，从未回到预训练水平；分布外成功率与对齐度的Spearman相关系数达0.964，成功rollout的对齐度显著高于失败rollout

- 仿真结果：π₀在LIBERO上平均成功率从89.3%提升至92.4%（+3.1%），SimplerEnv上从35.4%提升至41.7%（+6.3%）；SpatialVLA在SimplerEnv上从43.8%提升至51.0%（+7.2%），跨架构验证有效性

- 真实机器人结果：π₀内部分布任务平均成功率从51.3%提升至70.0%（+18.7%），分布外平均成功率从49.5%提升至71.0%（+21.5%），在需要空间和组合推理的位置、任务组合轴上提升最显著

- 消融实验：共享-私有分解、对比对齐、EgoHOD作为对齐目标、第10层中间层对齐均为有效设计，仅用对比对齐即可获得 baseline 以上的提升，加入分解后进一步提升性能
