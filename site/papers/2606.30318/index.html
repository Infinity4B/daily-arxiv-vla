Card 02 论文概述

论文概述

  • 提出名为 Chronos 的物理启发全历史框架，用于解决非马尔可夫长时域机器人操作问题
  • 指出当前主流的 VLA 和模仿学习策略依赖当前观测或短窗口（马尔可夫捷径），在依赖记忆的任务中会因观测别名而失败：相同观测在不同历史下需要不同动作
  • 核心思想是将观测历史从辅助上下文提升为策略动力学的潜在状态，结合全历史状态 token 序列与二阶物理启发的动作生成机制
Card 03 核心贡献

核心贡献

  • 将长时域模仿学习形式化为全历史状态 token 建模，设计记忆感知的感知-状态空间管道，实现每个物理时间步一个 token 的完整轨迹级时间信用分配
  • 提出物理启发的二阶薛定谔桥理论用于动作生成，将动作块视为广义坐标并通过加速度场进行细化，取代传统的一阶速度场或去噪范式，使动作更平滑且物理合理
  • 在 16 个模拟任务和 4 个真实世界实验中全面验证，在依赖记忆的操作任务上以极小参数量大幅超越强 VLA 基线
Card 04 方法描述

方法描述

  • 全历史状态 token 化：在每个物理控制步，将外部观测与本体感觉融合为单个状态代表性 token，token 序列长度与物理轨迹长度一一对应，避免空间与时间 token 混淆
  • 选择性状态空间模型（SSM）：使用 Mamba 风格的线性时间递归结构传播内部潜在状态，生成因果历史上下文，支持晚期模仿损失回传更新早期阶段表征
  • 隐式最大似然估计（IMLE）先验：基于历史上下文与当前状态 token 生成粗粒度多模态动作先验，避免模式平均
  • 二阶薛定谔桥细化：从薛定谔方程出发，经 Madelung 变换得到量子哈密顿-雅可比方程与连续性方程，引入高斯桥近似与 Kostin 耗散势，推导出稳定的加速度匹配损失来细化动作轨迹

Card 06 评估与结果

评估与结果

  • 评估环境与基准：涵盖精度插入、通用操作和记忆依赖的长时域控制等 16 个仿真任务及 4 个真实世界任务
  • 主要评估指标：平均成功率
  • 关键实验结果：在 RMBench 上达到 73.6% 平均成功率，比马尔可夫 VLA 基线 $\pi_{0.5}$ 高 +62.4 绝对百分点（6.6倍相对增益），参数量少 10倍；比记忆 VLA Mem-0 高 22.8 百分点，参数量少 30倍以上
