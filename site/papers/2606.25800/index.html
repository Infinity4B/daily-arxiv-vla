Card 03 核心贡献

核心贡献

  • 提出 ROAD-VLA 自蒸馏框架，通过优势估计扰动动作 logits 构建近端教师，将稀疏标量奖励转化为密集 token 级监督
  • 提供形式化分析，证明在标定条件下蒸馏至优势引导的近端教师具有策略改进保证
  • 在涵盖视觉鲁棒性、组合推理和执行鲁棒性三个维度的七个机器人操作环境中进行评测，ROAD-VLA 全面优于 PPO 基线
  • 通过消融实验验证文本引导特权信息的效果有限，而动作空间内的优势引导信号是有效的
Card 04 方法描述

方法描述

  • 使用 自蒸馏 策略：学生策略在线收集轨迹，利用优势估计构建近端教师分布，通过 KL 散度进行 token 级蒸馏
  • 设计 优势引导的特权上下文：将混合优势（内在估计与参考估计的符号一致加权）标准化并裁剪为扰动权重，对每个动作 token 的 logits 进行扰动得到教师分布
  • 采用 标定的一致门控：仅当内在与参考优势符号一致时混合，防止不匹配的 critic 噪声
  • 构建 近端教师：通过对数几率扰动等价于在 KL 正则化下最大化奖励的闭式解，保持教师靠近当前策略
  • 理论分析给出策略改进下界，表明教师-策略 KL 散度作为改进项，蒸馏误差作为代价项
Card 05 数据集与资源

数据集与资源

  • 基础模型：OpenVLA-7B
  • 评估环境：七个机器人操作环境，分为三类分布偏移

- 视觉鲁棒性：VR-UnseenTable, VR-DynamicTexture, VR-DynamicNoise

- 组合推理：CR-MultiObject, CR-MultiReceptacle

- 执行鲁棒性：ER-InitPose, ER-Repositioning

  • 训练资源：140 条专家演示用于初始预热微调，PPO 和 ROAD-VLA 共享相同 rollout 缓冲区和超参数
  • 实现细节：OpenVLA 使用 SigLIP-DINOv2 视觉编码器，Llama 2 tokenizer，动作 token 数为 7
Card 06 评估与结果

评估与结果

  • 评估环境：包含分布内（ID）和分布外（OOD）设置，每个环境 3 个随机种子
  • 主要指标：任务成功率（%），ID 与 OOD 成功率，以及性能退化 Δ = ID - OOD
  • ROAD-VLA 在所有环境平均：

- ID 成功率从 85% 提升至 88%

- OOD 成功率从 69% 提升至 73%

- 平均退化幅度从 16.3% 降低至 14.6%

  • 在 VR-DynamicTexture 和 ER-Repositioning 等困难场景下，退化幅度显著降低（22%→19%，16%→11%）
  • 在线适应过程中，ROAD-VLA 收敛更快，峰值更高，后期方差更小，得益于密集的 token 级监督和一致门控机制
