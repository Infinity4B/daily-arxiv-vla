MemoryWAM: Efficient World Action Modeling with Persistent Memory

2026-06-18 · 原文 · 翻译 · 2606.20562

论文提出 MemoryWAM，一个具备高效持久记忆的世界动作模型（WAM），旨在解决现有 WAM 在记忆能力与计算效率之间的根本性权衡。 现有 WAM 要么仅依赖有限的近期观测窗口而无法处理非马尔可夫任务，要么保留完整历史导致推理延迟和 GPU 内存随序列长度线性增长。 MemoryWAM 通过模拟人类混合记忆的混合记忆设计，在保持长期记忆的同时大幅降低推理成本。

Card 03 核心贡献

核心贡献

  • 提出 MemoryWAM，一个融合滑动窗口上下文、主旨令牌（gist tokens）和锚点帧（anchor frames）的混合记忆世界动作模型，实现高效的持久历史记忆。
  • 对世界动作模型的记忆机制进行了系统性研究，分析了全注意力、测试时训练（TTT）、循环神经网络（RNN）与所提混合记忆在延迟、内存和性能上的权衡。
  • 在长周期、记忆依赖的仿真和真实世界操作任务中，MemoryWAM 一致性地超越了最强的视觉-语言-动作（VLA）和 WAM 基线，同时保持了更优的计算效率。
Card 04 方法描述

方法描述

  • 采用 MoT（Mixture-of-Transformers）架构，包含一个视频 DiT（用于提取动态感知特征并维护记忆）和一个动作 DiT（根据视频特征预测动作块），训练时通过视频预测监督动态建模，推理时无需视频生成。
  • 提出 混合记忆机制，包含三种记忆组件：短期记忆（滑动窗口保留最近帧的高保真上下文）、事件边界记忆（保留任务初始锚点帧的完整视觉令牌）和主旨记忆（为每帧附加少量可学习的 gist tokens 来压缩长程历史）。
  • 通过精心设计的注意力掩码，后续帧和动作令牌仅关注近期帧、锚点帧的全量令牌以及历史帧的压缩 gist tokens，从而将长期记忆的 KV 缓存复杂度从 O(N) 降至 O(N/d)，压缩比 d=15。
Card 05 数据集与资源

数据集与资源

  • 使用仿真基准 RMBench（包含 9 个记忆依赖的双臂操作任务）进行训练和评估，所有方法均使用每个任务 50 个专家演示。
  • 真实世界实验包括 Shell Game 和 Look and Press 两个记忆依赖任务，硬件平台为 ARX 双臂机器人和 RealSense D455 相机。
  • 模型基于预训练的 Wan2.2-TI2V-5B 视频 DiT 构建，动作专家为 1B 参数，总参数量约 6B。
  • 训练使用 8 个 GPU，每 GPU 批次大小为 1，优化器为 AdamW。
Card 06 评估与结果

评估与结果

  • 在 RMBench 仿真基准上，MemoryWAM 取得 83.0% 的平均成功率，显著优于依赖短期记忆的 π0.5（10.4%）和 FastWAM（5.9%），并比全历史记忆的 LingBot-VA（78.2%）高出 4.8 个百分点。
  • 在真实世界实验中，MemoryWAM 在 Shell Game 和 Look and Press 任务上分别取得 18/20 和 15/20 的成功率，均优于 π0.5 和 LingBot-VA。
  • 效率对比显示，混合记忆机制在长序列下比全注意力机制显著降低了推理延迟和 GPU 内存使用，即使序列长度达到 1600 帧，仍比 RNN 和 TTT 方法更高效。
  • 消融实验表明，移除主旨令牌导致性能下降最大，移除锚点帧或滑动窗口也会降低性能，验证了混合记忆各组件的必要性。
