论文概述

  • 提出 Action ControlNet (ACNet)，一种轻量级延迟感知适配器，用于解决视觉-语言-动作（VLA）模型在异步控制时因推理延迟导致的动作不连续问题

  • 使 VLA 策略在无需全量重训练的情况下，提升异步部署时的鲁棒性与轨迹平滑性
Card 03 核心贡献

核心贡献

  • 将异步分块 VLA 控制中的延迟退化问题形式化为块交接处的边界条件问题，并识别出已执行动作后缀为关键条件信号
  • 提出 ACNet，一种轻量级的动作头级别残差适配器，仅引入少量可训练参数，可作为即插即用模块附加到现有分块 VLA 策略（如扩散、流匹配）
  • 在 Kinetix、Meta-World MT50 及真实 SO-ARM101 平台上验证了 ACNet 在保持较低适配成本的同时，显著提升延迟鲁棒性和轨迹平滑性
Card 04 方法描述

方法描述

  • 延迟动作编码：将上一个动作块在推理延迟期间已执行的后缀（延迟动作）与可学习填充 token 拼接，经轻量 Transformer 编码后进行时序池化，得到延迟上下文向量
  • 残差注入：将延迟上下文向量通过投影层以残差形式注入到冻结的动作专家网络中，优选注入到最后一层，实现局部边界校正，而不改变全局任务感知
  • 训练策略：冻结感知-语言主干，仅训练延迟编码器、投影层和端部池化层；在流匹配框架下，对同一视觉-语言潜变量采样多个延迟条件，提升训练效率
Card 05 数据集与资源

数据集与资源

  • 使用数据集：Kinetix 仿真器任务（基于 RTC 协议）、Meta-World MT50 全套 50 个任务
  • 基线模型：π₀（用于 RTC 和 Training-RTC 基线）、Evo-1（ACNet 的骨干网络）
  • ACNet 可训练参数占比约 20%，远低于完全延迟条件重训练（100% 参数更新）
  • 训练硬件：单张 RTX 4080 SUPER GPU
Card 06 评估与结果

评估与结果

  • 评估基准：Kinetix 和 Meta-World MT50 异步操控任务，比较 Naïve Async、RTC、Training-RTC 和 ACNet
  • 主要指标：任务成功率、可训练参数比例、端到端推理延迟、控制频率、动作 jerk（平滑性）
  • ACNet 在 Kinetix 的延迟条件下平均成功率 0.79，与 Training-RTC（0.80）相当，但仅更新约 20% 参数；在 Meta-World MT50 上平均成功率 0.74，推理延迟 91 ms，控制频率 11.0 Hz，均优于 RTC 和 Training-RTC
  • 真实机器人实验（SO-ARM100）中，ACNet 在两个任务上均达到 100% 成功率，且轨迹抖动显著低于 Naïve Async；消融实验证实最后一层注入和可学习填充 token 的设计最为有效
