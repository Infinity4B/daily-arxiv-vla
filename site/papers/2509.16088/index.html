Card 02 论文概述

论文概述

随机化平滑（Randomized Smoothing, RS）是目前在大规模分类模型上唯一可行的鲁棒性认证技术，但其原本只能处理离散标签，难以直接用于生成式的视觉‑语言模型（Vision‑Language Models, VLMs）。本文提出一种将生成式模型的输出映射为分类问题的通用方法，从而把 RS 迁移到 VLMs 场景。核心思路是引入一个“oracle”分类层，对模型生成的文本进行二分类（如有害/无害）或离散动作映射，或将语义等价的答案归为同一等价类。通过对图像添加高斯噪声、保持提示文本不变、进行多次采样并使用 oracle 进行投票，得到多数类的置信下界，进而推导出可验证的鲁棒半径。论文还从理论上分析了样本数对半径和认证精度的影响，给出改进的样本复杂度标度律，使得在实际应用中只需要 10²‑10³ 级别的样本即可得到与经典方法（10⁴‑10⁵ 样本）相匹配的证书。实验在最新的 VLM 上对最近的越狱式攻击进行了验证，展示了该方法的可扩展性和实用性。

Card 03 核心贡献

核心贡献

  • 将随机化平滑从分类扩展到生成式 VLM，提出基于 oracle 的抽象层，使得生成结果可以被视作离散类。
  • 给出在 oracle 错误率 ε<0.5 下的概率下界修正公式

\[

\bar p_y = \frac{\bar q_y - \epsilon}{1-2\epsilon}

\]

并证明在 \(\bar q_y>0.5\) 时该下界仍然是有效的。

  • 推导出样本数 \(n\) 与可验证半径的关系（利用中心极限定理和 Shore 对 \(\Phi^{-1}\) 的近似），得到标度律

\[

r_\sigma(\alpha,n) \approx 1 - 1.64\frac{z_\alpha}{\sqrt n}

\]

表明样本量只需降低 2‑3 个数量级即可保持接近最优的半径。

  • 在理论上放宽了前期工作对均匀分布的要求，仅需多数类概率分布主要集中在 \([\beta,1)\) 且 \(\beta\ge 0.7\)，从而提升了方法的适用性。
  • 给出适用于 VLM 的完整认证算法（Algorithm 2），包括语义等价聚类的投票机制。
  • 通过在最新 VLMs 上进行实验，验证了对 jailbreak‑style 攻击的防御能力，并报告了认证半径、认证精度以及样本效率的定量结果。
Card 04 方法描述

方法描述

  1. 模型与噪声注入

- 输入为图像 \(\mathbf x\) 与文本提示 \(\mathbf t\)。

- 对图像加高斯噪声 \(\mathbf z\sim\mathcal N(\mathbf 0,\sigma^2\mathbf I)\) 生成 \(\mathbf x'\)。

- 文本保持不变，调用 VLM \(f_\theta(\mathbf x',\mathbf t)\) 获得生成的回答 \(\mathbf y\)。

  1. Oracle 分类层

- 内容安全分类：oracle 将 \(\mathbf y\) 判为 “有害” 或 “无害”。

- 离散动作映射：若 VLM 充当 VLA，oracle 将答案映射为有限的动作集合（如 base‑forward、gripper‑open）。

- 语义等价聚类：oracle 判断新回答是否与已出现的回答语义相同，若相同则累计计数，否则创建新类。

  1. 投票与计数（Algorithm 2）

- 用字典 ans 存储每个等价类（或每个离散动作）的出现次数。

- 对每个噪声样本执行 oracle 判断并更新相应计数。

- 最终返回计数最高的回答 \(y\) 与其计数 \(c\)。

  1. 概率下界与证书半径

- 将计数 \(c\) 与样本数 \(n\) 带入 Clopper‑Pearson 方法得到 \(\bar q_y\)（在置信度 \(1-\alpha\) 下的下界）。

- 考虑 oracle 误差 \(\epsilon\)（假设 \(\epsilon<0.5\)），修正得到真实概率下界

\[

\bar p_y = \frac{\bar q_y - \epsilon}{1-2\epsilon}

\]

- 若 \(\bar p_y>0.5\)，则可验证半径为

\[

R = \sigma \Phi^{-1}(\bar p_y)

\]

- 对于二分类情形，若对 \(\epsilon\) 没有任何已知信息，只要 \(\bar q_y>0.5\) 仍可直接使用 \(R = \sigma\Phi^{-1}(\bar q_y)\) 作为下界（Theorem 4.2）。

  1. 样本效率分析

- 利用 CLT 对 \(\bar p_y\) 进行近似，得到期望半径

\[

R_\sigma^{\alpha,n}(p_A) \approx \sigma \Phi^{-1}\!\bigl(p_A - t_{\alpha,n}\bigr),\quad

t_{\alpha,n}=z_\alpha\sqrt{p_A(1-p_A)/n}

\]

- 通过 Shore 对 \(\Phi^{-1}\) 的幂级数近似，进一步得到平均半径的下降比例

\[

r_\sigma(\alpha,n) = \frac{\bar R_\sigma(\alpha,n)}{\bar R_\sigma(0,\infty)} \approx 1 - 1.64\frac{z_\alpha}{\sqrt n}

\]

- 该公式说明把样本数从 \(10^5\) 降到 \(10^3\) 仍能保持约 90% 以上的理想半径。

  1. 实验实现

- 采样若干噪声图像（如 \(n=500\)‑\(2000\)），对每个图像调用 VLM 并由强 LLM（如 GPT‑4、Llama‑70B）充当 oracle。

- 计算多数类的计数、Clopper‑Pearson 下界与对应半径。

- 与原始分类 RS 基线以及仅使用 Clopper‑Pearson 不进行 oracle 修正的方案进行对比。

Card 05 数据集与资源

数据集与资源

  • 模型：直接使用公开的预训练 VLM（如 LLaVA、InstructBLIP、GPT‑4V 等），不进行额外微调。
  • Oracle：使用更强的语言模型（例如 GPT‑4 或 Llama‑70B）作为分类器或语义等价判断器。
  • 实验数据：在论文正文中未提供详细数据集描述，附录 B 中列有用于内容安全、VLA 动作与语义聚类的图像‑提示对（来源于公开的 VLM 评测集），并对每个样本进行多次噪声采样。
  • 计算资源：实验基于多卡 GPU 集群执行 VLM 推理，oracle 的调用使用相同的 GPU 或 CPU 后端。由于模型已预训练，计算开销主要是多次前向推理（每张图像 \(n\) 次），具体硬件规格未在正文中给出。
Card 06 评估与结果

评估与结果

  • 评估环境：在标准机器学习服务器上运行（Python + PyTorch），所有 VLM 与 oracle 均通过相同的推理框架进行调用，实验代码已开源。
  • 鲁棒性指标

- 可验证半径 \(R\)：在给定置信度 \(\alpha\) 下，对每个输入返回的最大半径。

- 认证精度：在特定半径阈值（如 \(R=0.5\)）下，能够给出非“ABSTAIN”结果的样本比例。

- 半径下降比例：通过标度律或实际测量比较不同样本数 \(n\) 下的平均半径 \(\bar R_\sigma(\alpha,n)\)。

- Oracle 误差敏感性：评估在不同假设的 \(\epsilon\)（oracle 错误率）下半径下界的变化。

  • 对抗评估

- 对最新的 jailbreak‑style 攻击（Qi et al., 2024）进行防御测试，统计攻击成功率的降低程度。

- 与未加噪声的原始 VLM 对比，展示 RS 增强后的安全性提升。

  • 样本效率

- 记录在不同样本规模（如 \(n=100, 500, 1000, 5000\)）下的认证半径、认证精度以及所需的计算时间。

- 通过经验曲线验证理论标度律 \(1-1.64z_\alpha/\sqrt n\) 的准确性。
