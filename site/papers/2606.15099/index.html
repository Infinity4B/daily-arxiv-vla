Card 02 论文概述

论文概述

  • 现有视觉-语言-动作（VLA）模型普遍采用显式思维链（CoT）推理桥接感知与动作，存在计算开销大、多步任务错误累积传播的问题
  • 本文提出自适应变量对齐VLA（AVA-VLA）框架，将推理建模为不可观测的连续潜变量序列，无需显式文本生成即可完成推理过程
  • 研究目标是同时提升VLA模型的推理效率与长程任务决策稳定性，解决显式推理范式在实时机器人场景中的局限性
Card 03 核心贡献

核心贡献

  • 提出AVA-VLA框架，将内部推理过程建模为潜变量序列，绕过显式CoT的计算瓶颈与稳定性问题
  • 引入基于强化学习（RL）的推理去噪机制，将潜推理生成视为序列决策过程，通过任务级奖励优化潜轨迹，确保推理过程与下游任务目标对齐
  • 设计自适应早停策略，让模型可根据潜状态置信度动态终止推理，在“快思考”与“慢思考”之间灵活切换，平衡推理深度与计算效率
  • 在多个具身决策基准上验证方法的有效性，证明其可在显著降低推理延迟的同时，达到甚至超越全推理基线的性能
Card 04 方法描述

方法描述

  • 首先将潜推理过程形式化为部分可观测马尔可夫决策过程（POMDP），明确引入不可观测的潜推理状态$z_t$编码多模态信息的中间推理表示
  • 定义参数化推理策略$\pi_\phi$，根据当前潜状态与多模态观测生成潜更新动作$u_t$，控制潜状态的更新方向与强度
  • 通过参数化状态转移函数$f_\theta$实现潜状态的连续演化，保证推理过程的时间连贯性
  • 设计RL-based推理去噪模块：构建包含任务奖励、熵正则项、平滑正则项的复合奖励函数，采用PPO+GAE优化推理策略，通过长期任务信号约束潜轨迹的稳定性，减少噪声干扰与表示漂移
  • 提出自适应早停机制：通过门控网络$g_\omega$评估当前潜状态的决策置信度，当置信度超过校准阈值$\tau$时提前终止推理，直接生成动作，自适应裁剪冗余计算
Card 05 数据集与资源

数据集与资源

  • 使用数据集：LIBERO benchmark（含Spatial、Object、Goal、Long 4个任务子集，及引入7个扰动维度的LIBERO+鲁棒性测试版本）、CALVIN benchmark（采用ABC→D跨环境泛化设置，评估长程操作能力）
  • 模型规格：视觉特征维度768，语言嵌入维度768，推理与动作网络隐藏层大小分别为1024、2048，推理更新动作空间为$\mathbb{R}^{64}$的连续空间
  • 训练资源：使用8块NVIDIA A100 80GB GPU，PPO强化学习阶段共进行约120万环境交互步，总训练时长18.6小时
Card 06 评估与结果

评估与结果

  • 评估环境与基准：在LIBERO（分单策略覆盖所有任务、单策略 per 任务子集两种设置）、CALVIN ABC→D跨环境泛化基准上评估
  • 主要评估指标：任务成功率（SR，%）、平均任务完成长度、推理延迟（ms）、吞吐量（Hz）
  • 关键实验结果：

- LIBERO基准上，单策略设置下平均成功率达到98.3%，为所有对比方法最高；单策略 per 子集设置下，空间任务成功率99.6%、物体操作任务成功率99.7%

- CALVIN基准上，5步长程任务成功率达到84.0%，平均任务完成长度4.77，优于所有对比基线

- 效率层面，相比显式CoT方法实现6倍推理加速，早停机制将平均推理步数从5.0降低至2.3，平均延迟仅145ms，仅比最快的反射式基线$\pi_0$-FAST高47ms，同时长程任务（LIBERO-Long）成功率比最快基线高4.1%
