论文概述

  • 提出 EquiVLA，首个端到端 SO(2) 旋转等变 Vision-Language-Action (VLA) 模型框架，解决现有 VLA 模型在物体旋转泛化方面数据效率低、需大量训练数据的问题

Card 03 核心贡献

核心贡献

  • 提出 Token-level Frame Averaging (EquiPerceptor)，将 Frame Averaging 从全局池化向量扩展到空间索引的 ViT patch token 序列，从任意冻结 VLM 产生近似 SO(2)-等变视觉表示
  • 提出 EquiActor，首个 SO(2)-等变 flow-matching Diffusion Transformer action head，通过可操纵层 (steerable layers) 实现等变注意力、状态编码和动作解码
  • 建立端到端等变 VLA 框架 (EquiVLA)，结合 EquiPerceptor 和 EquiActor，无需修改预训练 VLM 权重即可实现等变映射
  • 在 GR00T N1.5 上验证，LIBERO 相对控制平均成功率提升至 92.6%（+14.5 pp），CALVIN 平均序列长度提升至 4.03（+0.58），真实机器人成功率从 54% 提升至 72%
Card 04 方法描述

方法描述

  • EquiPerceptor：对冻结 ViT 输出的 patch token 序列应用 Token-level Frame Averaging，通过空间排列变换 τ(h⁻¹) 和正则表示特征变换 ρ_reg(h⁻¹) 将旋转后的 token 映射回规范位置后平均，分离出等变流 z^eq 和不变量流 z^inv；通过等变适配器 (Equivariant Adapter) 融合等变视觉特征与 VLM 语言-视觉上下文特征，适配器使用不变量门控混合语义分支与等变分支
  • EquiActor：将标准 DiT action head 替换为完全 SO(2)-等变版本，所有线性投影、Q/K/V 注意力矩阵、状态/动作编码器和动作解码器均采用 G-可操纵层（正则表示到正则表示），注意力分数通过几何内积 ⟨q,k⟩=Σ_g q[g]·k[g] 计算以保持等变性
  • 等变动作表示：末端执行器位置和方向作为 SO(2) 向量（使用 ρ₁ 和 ρ₂ 表示分解），夹爪宽度作为旋转不变量 (ρ₀)，支持绝对控制和相对控制两种模式
  • 对称群：采用 C₈（8重离散旋转对称群）作为近似 SO(2) 的有限子群，消融实验验证了 C₄、C₈、C₁₆ 的选择
Card 05 数据集与资源

数据集与资源

  • LIBERO 基准：包含 LIBERO-10、LIBERO-Goal、LIBERO-Object、LIBERO-Spatial 四个套件，每个套件 10 个任务，每任务约 40 条演示
  • CALVIN 基准：ABCD→D 长程任务链基准，5 个连续语言条件操作任务，训练于 A/B/C/D 环境，零样本测试于环境 D
  • 真实机器人：Mobile ALOHA 平台，5 个桌面操作任务（Banana in Pot、Block Storing、House Building、Letter Aligning、Shorts Folding），每任务 150 条遥操作演示
  • 模型基础：GR00T N1.5（冻结 VLM 骨干网络 + 可训练的等变 action head）
  • 训练资源：所有模型共享相同预训练权重和超参数（具体 GPU/TPU 配置未在截取部分明确说明）
Card 06 评估与结果

评估与结果

  • 评估环境：LIBERO（相对/绝对控制，每任务 50 次 rollout，2 个随机种子）、CALVIN（1000 条指令链）、Mobile ALOHA（每任务 20 次试验）
  • 主要指标：任务成功率、长程任务序列长度、等变误差 ε_eq
  • 关键实验结果：

- LIBERO 相对控制：EquiVLA 平均 92.6%（GR00T N1.5 基线 78.1%，π₀ 86.0%），绝对控制 76.1%（基线 62.6%）

- CALVIN：EquiVLA 平均序列长度 4.03（GR00T N1.5 3.45，接近多帧方法 MoDE 的 4.39），第 5 个任务成功率从 48.5% 提升至 64.3%（+15.8 pp）

- 真实机器人：平均成功率 72%（GR00T N1.5 54%），Letter Aligning 从 65% 提升至 95%（+30 pp），House Building 从 15% 提升至 50%（+35 pp）；Shorts Folding 两者均为 85%，说明等变偏置在无旋转对称任务中无额外成本

- 等变误差：基线 7.754±3.572，EquiActor 降至 0.837±0.738（9.3×），EquiVLA 进一步降至 0.284±0.134（27.3×）

- 样本效率：在 10% 演示数据下高 1.8 pp，40% 数据下高 10.6 pp，100% 数据下高 14.5 pp，证明等变归纳偏置在低数据 regime 下最有益
