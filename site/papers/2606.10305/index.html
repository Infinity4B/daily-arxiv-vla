Card 02 论文概述

论文概述

  • 提出 SARM2，一种多任务阶段感知奖励模型，通过结合动作基元阶段估计器与多门控混合专家（MMoE）值头，为长程机器人操作任务生成稠密、准确且通用的奖励信号
  • 提出 SPIRAL（Self-Policy Improvement via Reward-Aligned Learning），一个基于SARM2稠密奖励的自主机器人自我改进框架，利用廉价自主rollout实现策略的持续优化与数据飞轮
  • 解决现有方法的核心局限：任务特定阶段感知模型依赖每任务人工标注，而通用VLM奖励模型过于粗糙，无法为长程任务提供细粒度时序监督
Card 03 核心贡献

核心贡献

  • SARM2：首个将MoE应用于多任务机器人奖励模型的工作，通过任务无关的动作基元阶段估计器与MMoE值解码器的紧密耦合，在单一模型中跨多样化长程任务产生准确稠密奖励
  • SPIRAL：一种奖励对齐的在线策略改进框架，结合DICE-RL的分布收缩RL微调与稠密奖励监督，通过rollout-重标注-更新的闭环实现无需人工介入的自主策略提升
  • 在10任务基准和2个真实世界长程任务（Folding Shorts与Cleaning Whiteboard）上验证，SARM2将价值估计MSE降低80%，策略成功率从约50%提升至近100%
Card 04 方法描述

方法描述

  • Action-Primitive Stage Estimator：将阶段估计建模为21个动作基元+1个null类的时间序列分类，使用4层因果Transformer处理6帧历史（步长30）与本体感觉输入，采用冻结的SigLIP-2编码器提取多视角视觉特征
  • MMoE Value Decoder：采用6层因果Transformer，根据预测的动作基元选择对应的多门控混合专家路由，将22个基元分组为M+1个语义组，通过Top-k路由激活共享专家池，预测归一化剩余完成步数作为稠密进度奖励
  • SPIRAL框架：基于DICE-RL的残差RL方法，结合稠密奖励TD3目标与蒙特卡洛稀疏奖励目标，采用5个集成Q函数缓解过估计，通过多latent采样与best-of-k动作选择实现稳定探索与分布收缩
Card 05 数据集与资源

数据集与资源

  • 训练数据：从100个任务中策划的200小时真实世界操作数据，提取21个覆盖>90%任务时长的动作基元，构建66小时平衡数据集D_AP用于阶段估计器训练
  • 模型规模：阶段估计器约4层Transformer，值解码器约6层Transformer+MMoE，共享冻结SigLIP-2视觉编码器，总参数量约200M级别（远小于4B的Robometer）
  • 硬件配置：真实机器人实验（详见附录6.1），通过冻结视觉编码器缓存嵌入消除推理时冗余视觉编码
Card 06 评估与结果

评估与结果

  • 评估环境：10任务基准（S1: 5个经典任务如叠衣服、插拔电源；S2: 5个非常规任务如清洁白板、收伞），以及2个真实世界长程任务Folding Shorts和Cleaning Whiteboard
  • 评估指标：演示集MSE（越低越好）、Rollout分类分数ρ（越高越好）、MoE路由密度、任务成功率（SR）、平均进度百分比
  • 关键结果：

- SARM2在演示集MSE上达到0.006（S1）、0.031（S2）和0.020（整体），分别比最强基线ReWiND（0.036）和TOPReward（0.107）降低80%以上

- 在Folding Shorts任务上，SPIRAL+SARM2将Flat任务成功率从BC的1/12提升至12/12，Crumpled任务从0/12提升至8/12

- 在Cleaning Whiteboard任务上，成功率从4/20提升至18/20，平均进度从47.5%提升至97.5%

- 消融实验验证阶段感知、多门控路由和训练分布多样性均对性能有显著正向贡献
