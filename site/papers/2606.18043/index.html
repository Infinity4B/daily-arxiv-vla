Card 03 核心贡献

核心贡献

  • 引入了基于速度场分歧（VFD）的流匹配模型认知不确定性估计器，并提供了严密的数学理论推导（Theorem 4.1）。
  • 提出了 SAVE 框架，利用 VFD 引导基于流的 VLAs 进行主动多任务微调，优先为最不确定的任务和初始状态请求专家演示。
  • 实验证明了 VFD 在预测任务性能和检测部署失败方面优于基线方法，且 SAVE 在多任务适应中所需的专家数据量比基线减少至少 22 倍。
Card 04 方法描述

方法描述

  • 核心技术为速度场分歧（VFD），通过计算集成中各模型在学习速度场上的缩放平方差来近似分布间的 KL 散度，从而量化认知不确定性。
  • 理论推导表明，速度场差异在流匹配的后期时间步（噪声较低）权重更大，权重因子 $\kappa_s = s/(1-s)$。
  • SAVE 框架流程：从基础 VLA 训练集成 -> 计算候选初始观测的 VFD -> 在任务级别聚合不确定性均值 -> 使用带温度参数的分类分布采样任务 -> 选择该任务中最不确定的初始状态请求专家演示 -> 使用新旧数据混合微调集成模型。
