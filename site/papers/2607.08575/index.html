Card 02 论文概述

论文概述

  • 提出 FabriVLA，一个轻量级的视觉-语言-动作（VLA）模型，用于精确的多任务机器人操作
  • 旨在解决大规模 VLA 模型计算成本高、推理延迟大，难以满足实时机器人控制需求的问题
  • 通过紧凑的 1B 规模 VLM 骨干网络，在不依赖数十亿参数模型的情况下，实现有竞争力的操作性能
Card 03 核心贡献

核心贡献

  • 提出 FabriVLA 轻量级 VLA 架构，在 Meta-World MT50 基准上达到 90.0% 的层级平均成功率
  • 设计了 门控自注意力 机制，通过从零初始化的可学习门控参数，逐步学习动作序列间的时序依赖关系，提供平滑的优化路径
  • 提出 浅层 VLM 层融合 技术，将深层语义特征与浅层空间细节特征相结合，显著提升需精确空间定位的任务性能
  • 采用 单阶段联合训练 策略，从预训练 VLM 和随机初始化的动作头开始，对整个模型进行端到端优化
Card 04 方法描述

方法描述

  • 使用 InternVL3.5-1B 作为视觉-语言骨干网络，处理 448×448 的 RGB 图像和任务指令文本
  • 动作头采用 流匹配 生成框架，通过预测速度场将随机噪声逐步转换为动作序列
  • 动作头包含 8 个 FabriVLA TransformerBlock 层，每层执行门控自注意力、交叉注意力和带时间条件的前馈网络
  • 浅层融合选择骨干网络的第 6 层和第 14 层特征，通过可学习的线性投影进行拼接融合
  • 推理时采用 50 步的欧拉积分，从随机噪声生成 50 步的动作预测序列
Card 05 数据集与资源

数据集与资源

  • 训练数据集：Evo-1 Meta-World 演示数据集，包含 50 个任务、每任务 50 条轨迹，共 2500 条轨迹
  • 模型规模：约 0.89B 参数
  • 骨干网络：InternVL3.5-1B（14 层 Transformer）
  • 训练资源：5 块 NVIDIA RTX PRO 6000 GPU
  • 训练步数：100,000 步，使用 BF16 混合精度和 DeepSpeed ZeRO-2 加速
Card 06 评估与结果

评估与结果

  • 评估基准：Meta-World MT50，包含 50 个操作任务，分为 Easy、Medium、Hard、Very Hard 四个难度等级
  • 评估指标：层级平均成功率（Tier-Average）和整体回合级成功率（Overall Episode-Level）
  • 主要结果：层级平均成功率达 90.0%，整体回合级成功率 92.0%，在 Easy 层级达到 95.0%
  • 消融实验表明，浅层 VLM 层融合 将层级平均成功率从 82.9% 提升至 90.0%；门控自注意力 是动作头中的决定性组件，将其加入基础头后层级平均成功率从 48.5% 提升至 57.7%
