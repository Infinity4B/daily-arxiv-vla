Action Emergence from Streaming Intent

Card 03 核心贡献

核心贡献

  • 正式定义了端到端自动驾驶的 action emergence 能力，并指出 driving intent 是实现该能力的关键缺失组件。
  • 提出了 Streaming Intent 机制，包含语义流式（通过四步思维链 CoT 推导意图）和时间流式（通过记忆 Token 跨片段传递意图），并实例化为 SI 模型。
  • 在 Waymo End-to-End 基准上取得了有竞争力的性能（验证集 RFS 7.96，测试集 RFS 7.74）。
  • 首次在完全端到端的 VLA 模型中展示了 intent-faithful controllability（意图忠实可控性），无需预构建轨迹库或人工后处理选择器。
  • 提出了一种自动化数据构建流程，利用 VLM 生成密集的思维链监督和意图标签。
Card 04 方法描述

方法描述

  • 模型架构基于单一的 Qwen3-VL Transformer 主干，同时支持自回归（AR）语言分支和流匹配（FM）动作分支。
  • Streaming Intent 机制包含两部分：语义流式通过 Perceive -> Predict -> Judge -> Plan 四步思维链推导意图；时间流式通过前向意图记忆 Token 保持跨片段的意图一致性。
  • 解码出的意图 Token 通过 Classifier-Free Guidance (CFG) 驱动流匹配轨迹生成，使轨迹能够被明确的意图信号引导而非依赖数据先验。
  • 引入了 CFG 蒸馏技术，将原本需要两次前向传播的 CFG 采样压缩为单次传播，在不牺牲规划质量的前提下减半推理成本。
Card 05 数据集与资源

数据集与资源

  • 使用 Waymo Open Dataset End-to-End Camera (WOD E2E CAM v1.0.0) 数据集。
  • 模型主干为 Qwen3-VL-2B-Instruct，总参数量 2.46B，其中 2.07B 参数可端到端训练。
  • 训练数据包含 20,745 个驾驶片段，通过自动化流程标注了 20 类意图和 14 类元动作。
  • 训练使用 8 个 GPU，训练时长为 75 个 Epoch。
Card 06 评估与结果

评估与结果

  • 评估基准为 Waymo End-to-End (WOD-E2E) 挑战赛。
  • 主要指标为 Rater Feedback Score (RFS)、Trust Region Rate (TR) 和 Average Displacement Error (ADE)。
  • 在验证集上达到最高 RFS 分数 7.96，在测试集上达到 RFS 7.74 和 ADE 3s 1.24m / ADE 5s 2.81m。
  • 实验展示了在长尾场景下的 action emergence 能力，SI 能生成多样的意图忠实轨迹，避免了基线模型（如 RAP）的模式坍塌问题。
  • 证明了意图引导轨迹的高质量：不同意图生成的轨迹与人类评分的高分备选方案相吻合。
