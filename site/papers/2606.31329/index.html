Card 02 论文概述

论文概述

  • 现有的分层视觉语言动作 (VLA) 模型使用 2D 轨迹指导底层策略，但在与基于 3D 点云的策略对接时存在表示错位，导致轨迹紧贴场景表面（"涂鸦效应"）而非自由穿过空间。
  • 提出 3D HAMSTER 框架，通过让 VLM 规划器直接输出度量可靠的 3D 轨迹，弥合 2D 规划与 3D 控制之间的表示鸿沟。
  • 增强了 VLM 的深度编码器和密集深度重建目标，以实现度量一致的 3D 路径点预测，从而在分布偏移下显著提升操作鲁棒性。
Card 03 核心贡献

核心贡献

  • 提出一个分层 VLA 框架，通过规划器直接输出 3D 轨迹，弥合了 2D-3D 表示鸿沟，使轨迹可直接被基于点云的底层策略使用。
  • 提出一种训练配方，结合深度编码器、密集深度重建损失和精选数据混合，使预训练 VLM 能够生成度量 3D 轨迹。
  • 在轨迹预测、仿真和真实世界操作三种设置中进行全面验证，证明在 3D 度量空间中对齐规划与执行可在 2D 指导失效的分布偏移下实现鲁棒操作。
Card 04 方法描述

方法描述

  • 3D 轨迹规划器: 基于 Qwen3-VL-8B，引入专用的深度编码器（基于 LingBot-Depth 初始化）处理深度图，特征与 RGB 特征逐元素融合后输入 LLM 生成 $(u,v,d)$ 序列。
  • 深度重建损失: 在自回归轨迹预测损失之外，加入密集深度重建损失 ($L_{depth}$)，使用 L1 损失约束深度解码器重建全深度图，防止深度特征在微调中退化。
  • 两阶段训练: 阶段 1 对齐深度特征（冻结主干网络，仅训练深度投影器和解码器）；阶段 2 微调任务（冻结编码器，LLM 使用 LoRA 微调，训练投影器和解码器）。
  • 轨迹条件化底层策略: 基于 3DFA，将预测的 3D 轨迹反投影到世界坐标系并附加到场景点云中，加入可学习的模态嵌入区分轨迹与场景点，并确保轨迹点在点云下采样中不被丢弃。
Card 05 数据集与资源

数据集与资源

  • 3D 能力训练数据: RLBench (Sim), DROID (Real), InternData-M1 (Sim), RefSpatial (Mix)
  • 2D 保留训练数据: RoboPoint, PixMo, LVIS, Honey-1M
  • 评估基准: DroidSpatial-Bench (新提出的轨迹预测基准), Colosseum (仿真基准), 真实世界 Franka Panda 机械臂任务
  • 模型规模: VLM 规划器基于 Qwen3-VL-8B-Instruct (8B 参数)
  • 训练资源: 规划器训练使用 8x H100 GPUs，底层策略训练使用 4x H100 GPUs
Card 06 评估与结果

评估与结果

  • 轨迹预测精度: 在 DroidSpatial-Bench 上，3D HAMSTER 在 10cm Both 指标上达到 65.5%，显著优于 Gemini-3.0-Pro (29.7%) 和 RoboBrain-2.5-8B (60.1%)。
  • 仿真鲁棒性: 在 Colosseum 基准上，3D 轨迹指导的平均成功率 (44.8%) 高于 2D 指导 (38.8%) 和无指导 (36.6%)；在光照和纹理变化等外观扰动下优势最大。
  • 真实世界操作: 在真实机械臂任务中，3D 指导的平均成功率（按钮按压 80%，倾倒 68%，抓放 62%）超越 2D 指导和单体 VLA $\pi_{0.5}$；在视觉和空间分布偏移下退化最平缓，避免了 2D 指导的"涂鸦效应"。
