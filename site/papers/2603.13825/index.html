Building Explicit World Model for Zero-Shot Open-World Object Manipulation

2026-03-14 · 原文 · 翻译 · 2603.13825

提出了一个基于显式世界模型的框架，用于开放世界物体操作，通过构建环境的物理基础数字孪生实现零样本泛化。 该框架整合了开放集感知、数字孪生重建以及交互策略的采样与评估，旨在解决VLA模型依赖大规模演示数据且泛化能力受限的问题。 核心目标是在无需任何任务特定动作演示的情况下，使机器人能够对新物体和新任务进行零样本操作。

Card 03 核心贡献

核心贡献

  • 提出了一个基于显式世界模型的操作框架，实现了对新颖刚体物体和任务规格的零样本泛化，无需任务特定训练或演示。
  • 设计了一个动态数字孪生构建流水线，能够从机载观测中重建物体网格，并对齐尺度与姿态以实现仿真一致的真实世界迁移。
  • 开发了一个基于视觉-语言模型的评估模块，用于评估采样动作候选的模拟结果的成功概率。
Card 04 方法描述

方法描述

  • 框架包含四个主要模块：开放集分割与抓取、数字孪生构建、操作策略采样。
  • 采用 GPT-4o 与 Grounded-SAM 进行开放集物体分割，使用 AnyGrasp 进行抓取姿态预测。
  • 创新性地提出了两阶段姿态对齐管道：首先利用 DINOv2 特征相似度进行粗对齐，再结合 RANSAC 和 ICP 进行精细对齐，解决了生成网格与真实观测的配准问题。
  • 在 NVIDIA Isaac Sim 仿真环境中采样多样化的交互策略，并利用 GPT-4o 和 高斯过程分类器 评估模拟结果以选择最优动作。
Card 05 数据集与资源

数据集与资源

  • 未使用特定的公开数据集，主要依赖真实机器人实验验证。
  • 使用的关键模型包括：GPT-4o (VLM)、Grounded-SAM (分割)、Hunyuan3D 2.0 (3D生成)、DINOv2 (特征提取)、AnyGrasp (抓取预测)。
  • 硬件资源：一台配备 NVIDIA RTX 2070 GPU (6GB RAM) 的笔记本用于仿真，一台配备 NVIDIA RTX 3090 GPU (24GB RAM) 的工作站用于计算机视觉模型。
  • 机器人平台：Kinova Gen3 Lite 7自由度机械臂，配备两个 Intel RealSense D405 相机。
Card 06 评估与结果

评估与结果

  • 实验分为数字孪生对齐精度评估与真实机器人任务性能评估两部分。
  • 对齐评估指标：对齐成功率与均方根误差。实验表明，所提出的两阶段对齐方法相比直接使用 RANSAC/ICP 的基线方法，显著提高了成功率，同时保持了相当的几何精度。
  • 真实机器人实验：设计了9个多样化的开放集操作任务，共进行96次试验。其中6个任务的成功率达到75%以上，部分任务（如“将黄色立方体放入白色盒子”）成功率达91.7%。
  • 主要失败原因在于基于LLM的结果检查器对视觉模糊性敏感，以及仿真到现实的差距（物理引擎精度、硬件控制延迟、传感器噪声等）。
