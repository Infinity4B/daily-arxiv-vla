Card 02 论文概述

论文概述

  • 提出 TacCoRL 框架，将触觉反馈（Tactile Feedback）集成到预训练的 Vision-Language-Action (VLA) 策略中，通过 Sim-Real 协同训练 和基于模拟的 强化学习（RL） 进行后训练改进
  • 解决接触丰富任务（contact-rich tasks）中视觉观察无法提供局部接触状态（如对齐、接触位置、压力分布）的问题，且无需大规模触觉预训练或 extensive real-world contact exploration
  • 使用真实对齐的模拟器作为闭环训练环境，学习接触读数应如何调节近故障状态（near-failure states）的动作响应，这些状态在演示中罕见且在硬件上收集风险较高
Card 03 核心贡献

核心贡献

  • 触觉条件接口：无需大规模触觉预训练或从头训练触觉策略，通过添加触觉编码器和接触感知门控机制即可适应预训练 VLA 策略
  • Sim-to-Real 后训练流水线：结合 Sim-Real 协同训练与基于可验证任务奖励的模拟器 RL，并由真实数据锚定（real-data anchor）学习可迁移的接触引导动作精炼
  • 真实机器人验证：在四个双手接触丰富任务上实现真实世界部署，平均成功率显著优于纯视觉和纯模仿基线
  • 接触感知门控机制：基于 taxel 读数动态抑制非接触时的触觉 token 影响，仅在检测到有效接触时激活触觉融合路径
Card 04 方法描述

方法描述

  • 基础架构：基于 π0.5 风格的 VLA 模型，采用条件流匹配（conditional flow matching）损失进行动作预测
  • 触觉编码与融合：触觉编码器 $E_\tau$ 处理最近 $L$ 帧触觉历史窗口，通过投影矩阵 $W_\tau$ 生成 $M$ 个触觉 token；采用交叉注意力机制（Cross-Attention）将触觉信息融合到视觉-语言-本体感知上下文，同时保留触觉 token 作为动作专家的条件输入
  • Sim-Real 协同训练：混合模拟（通过 MimicGen 扩展）和真实演示数据，使用加权流匹配损失 $\mathcal{L}_{\mathrm{co}}$，权重 $\alpha$ 控制模拟数据比例，为策略提供触觉条件动作先验
  • 后训练精炼：在真实对齐的模拟器中使用 PPO 进行稀疏奖励强化学习，优化接触行为；同时加入真实数据行为克隆损失（权重 $\beta$）作为锚定，防止策略偏离真实机器人分布
Card 05 数据集与资源

数据集与资源

  • 数据收集：每个任务收集 50 个真实世界演示和 20 个模拟遥操作轨迹，使用 MimicGen 将模拟集扩展到 200 条轨迹
  • 评估设置：使用 128 个并行模拟环境进行 RL 训练，每个任务进行 20 次真实世界试验评估
  • 基础模型：π0.5 预训练 VLA 模型（视觉-语言-动作流模型）
  • 任务场景：四个双手接触丰富任务，包括 Test Tube Insertion、Do Puzzle、Assembly #1 和 Assembly #2，配合域随机化（domain randomization）
Card 06 评估与结果

评估与结果

  • 评估基准：四个双手接触丰富操作任务，对比视觉-only策略与视觉-触觉策略
  • 主要指标：任务成功率（success rate）
  • 关键实验结果：

- 模拟器 RL 后训练：视觉-触觉策略平均成功率提升至 78.5%，显著高于视觉-only策略的 60.5%；直接从头 RL 训练基 VLA 模型在所有任务上均失败（0.0%），证明协同训练初始化的重要性

- 真实世界评估：触觉反馈将 RL 后训练策略的平均成功率从 50.0% 提升至 72.5%，证明接触感知增益来自触觉传感而非仅来自模拟器 RL

- 困难任务表现：在 Do Puzzle（最长时域任务）中，视觉-触觉策略达 45% 成功率，远高于真实-only微调（15%）和 Sim-Real 协同训练（25%）

- 超参数消融：协同训练比例 $\alpha=0.5$ 和锚定权重 $\beta=0.1$ 或 $1.0$ 效果最佳；过强锚定（$\beta=5.0$）会抑制 RL 精炼，导致性能下降；完全移除模拟数据（$\alpha=0$）导致策略与模拟器交互分布不匹配，性能骤降
