Card 02 论文概述

论文概述

  • 论文首次探索了Vision-Language guided Egocentric 3D Hand Pose Forecasting (VL-EHPF) 任务，旨在基于第一人称视觉观测、语言指令和手部姿态状态，预测未来时间步的第一人称3D手部关键点（每只手21个关键点，共42个，包含3D坐标），为增强现实、机器人操控等应用提供技术支撑
  • 指出现有方法存在两类核心局限：一类仅依赖视觉输入预测粗粒度动作，缺乏语言、细粒度姿态等完整任务上下文；另一类遵循VRM/VLA范式，受限于机器人数据规模不足、人机embodiment差异，难以有效迁移人类动作知识
  • 针对第一人称视角视野有限、手部动作动态性强的固有挑战，提出Exo2EgoPose框架，创新性地利用配对的 holistic 稳定的第三视角（Exocentric, Exo）示范视频作为引导，弥补第一人称视角的局部、动态信息不足
Card 03 核心贡献

核心贡献

  • 首次提出VL-EHPF任务与Exo2EgoPose框架，是首个探索利用第三视角示范视频弥补第一人称手部姿态预测信息不足的研究
  • 设计Dual-level Exocentric Reconstruction Module (DERM)，通过重建视频级和分块帧级的第三视角表示，建模第一人称场景的复杂时空上下文；设计Global-to-Local Modulation Module (GLMM)，利用分层第三视角表示，通过注意力机制和自适应调制单元（AMU）实现渐进式全局到局部的特征细化，提升预测精度
  • 构建新基准EgoMe-pose，在AssemblyHands、Ego-Exo4D、EgoMe-pose三个基准上均大幅超越现有最优方法（SOTA），同时验证了框架有效的人机知识迁移能力，在CALVIN机器人数据集上取得明显性能提升
Card 04 方法描述

方法描述

  • 首先采用多模态分词模块：分别用文本编码器、MAE视觉编码器、DINOv2视觉编码器、姿态编码器提取语言、第一人称视觉、手部姿态的特征，经MLP投影到统一表示空间，添加时间嵌入保留时序顺序，输入Transformer架构进行推理
  • 核心创新之一是DERM模块：训练阶段引入与第一人称视频配对的第三视角示范视频作为监督，通过可学习的查询重建第三视角的视频级（全局时序运动模式）和分块帧级（未来帧的细粒度空间上下文）表示，构建第一-第三视角的跨视角对应关系，建模第一人称场景的时空动态
  • 核心创新之二是GLMM模块：将重建的分层第三视角表示作为引导，先通过全局调制模块（GMM）利用视频级第三视角表示，经自适应调制和交叉注意力校准全局特征分布、捕捉长程时序动态；再通过局部调制模块（LMM）利用分块帧级第三视角表示，经交叉注意力和自适应调制实现帧级软对齐与细粒度特征校准，最终由MLP输出未来3D手部姿态和关节有效性掩码

数据集与资源

Card 06 评估与结果

评估与结果

  • 评估环境：在AssemblyHands（15fps）、Ego-Exo4D（30fps）、EgoMe-pose（15fps）三个基准的测试集上进行手部姿态预测任务评估，在CALVIN数据集上进行人机知识迁移验证
  • 主要评估指标：MPJPE（平均每关节位置误差，越低越好）、MPJVE（平均每关节速度误差，越低越好），关节有效性预测采用二分类交叉熵（BCE）损失
  • 关键实验结果：1. 在三个手部姿态预测基准的测试集上，所提方法均大幅超越现有最优方法（SOTA），定量结果显著优于USST、GCBC、MCIL等基线方法；2. 在CALVIN机器人数据集上的实验验证了框架有效的人机知识迁移能力，可实现人类手部动作知识向机器人操控任务的高效迁移，取得明显的性能提升；3. 消融实验验证了DERM和GLMM两个核心模块的有效性，定性分析显示模型能够准确预测未来手部姿态，特征调制机制可有效校准特征分布；4. 超参敏感性分析与模型复杂度分析表明所提框架具有较好的稳定性和效率
