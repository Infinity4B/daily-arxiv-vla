Card 02 论文概述

论文概述

  • 介绍了 WireCraft，一个用于工业 可变形线性物体 (DLO) 操控的仿真基准测试
  • 解决了现有基准缺乏模块化工业固定装置（连接器、夹子、通道）且难以对齐仿真与现实世界数据及评估协议的问题
  • 涵盖三个工业任务族：连接器插入、夹子布线和通道就位
Card 03 核心贡献

核心贡献

  • 提出了一个统一的仿真基准，包含可配置难度与资产的工业 DLO 操控任务
  • 提供了两种互补的 DLO 物理模型（Articulated 和 FEM-based Deformable），以平衡仿真速度与保真度
  • 在共享评估协议下对 RL、IL 和 VLA 策略进行了基准测试
  • 将公开发布仿真基准、轨迹数据、数据生成工具及 3D 可打印任务板组件
Card 04 方法描述

方法描述

  • 仿真后端基于 Isaac Lab v2.2.1 和 Isaac Sim 4.5 构建，支持 GPU 加速的并行训练
  • Articulated wire 使用刚体动力学（6-DoF 关节）实现高吞吐量；Deformable wire 使用 PhysX FEM 实现高保真度接触变形
  • 支持三种机器人实体：UR5、Franka 和 Trossen Stationary AI
  • 采用四种数据收集方法：脚本策略、RL 滚动、仿真遥操作和 真实世界 UR5 遥操作，并统一为 LeRobot 数据格式
Card 05 数据集与资源

数据集与资源

  • 使用共享的 LeRobot-compatible 数据集模式，包含 7 维状态/动作、同步 RGB 流和语言指令
  • 包含可配置的连接器资产：USB-A、DisplayPort、Cylinder、Cuboid 和 Ethernet
  • 提供任务侧固定装置（夹子、通道、母端口）的 3D 可打印模型以支持 Sim-to-Real 转移
  • 利用 Isaac Sim 的 GPU 加速多实例训练进行大规模策略学习与轨迹生成
Card 06 评估与结果

评估与结果

  • 在 Ethernet、Cylinder、Cuboid 和 DisplayPort 连接器插入任务上评估了 State PPO、Vision PPO、ACT、Diffusion Policy 和 $\pi_{0.5}$ 等基线
  • State PPO 在所有插入任务中达到 >92% 的成功率，在 1-clip 布线中达到 95.32%，在直通道就位中达到 82.33%，证实任务设定合理
  • Vision-based policies 存在显著的“到达-插入”差距，接触丰富对齐是主要瓶颈（例如 Vision PPO 在 Ethernet 插入中仅为 17.74%）
  • VLA 策略 $\pi_{0.5}$ 表现最差，到达与插入成功率仅为 6.2% 和 0.0%
  • 真实世界 UR5 验证表明，混合真实与仿真演示数据 (R+S) 训练的 ACT 实现了最佳物理插入成功率 (4/10)，而纯仿真数据无法零样本迁移
