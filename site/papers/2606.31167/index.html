Card 03 核心贡献

核心贡献

  • 提出 双尺度时间记忆枢纽（Dual-scale Temporal Memory Hubs），将长期场景布局和短期运动趋势压缩为固定长度的提示，克服时间近视且不增加上下文窗口负担。
  • 提出 潜在推理令牌（Latent Reasoning Tokens），通过互信息目标进行优化，在多模态上下文与动作轨迹之间建立语义规划空间，避免昂贵的文本监督。
  • 提出 并行动作解码方案（Parallel Action Decoding），用向量级预测替代标量级自回归生成，最大化控制吞吐量并减少解码延迟。
  • 在开源仿真基准和真实世界平台上取得了 SOTA 性能，并展现了显著的错误恢复能力。
Card 04 方法描述

方法描述

  • 时间记忆枢纽：包含长期工作空间枢纽（使用多尺度指数移动平均 EMA 融合场景特征与运动统计量）和短期记忆枢纽（使用时间注意力池化近期帧队列），两者通过 Sigmoid 门控融合，并以前缀或注入方式整合至当前帧特征中。
  • 潜在推理：在多模态上下文与动作令牌之间插入可学习的推理令牌，利用 InfoNCE 损失 构建互信息目标，同时约束推理表征与动作表征及上下文表征的对齐，无需人工文本标注即可形成规划桥梁。
  • 动作解码：为每个完整动作向量分配单个令牌，通过轻量级两层投影头在一次前向传播中并行解码整个动作轨迹，训练损失结合了 L1 回归损失与互信息推理损失。
Card 05 数据集与资源

数据集与资源

  • LIBERO 仿真基准数据集（包含 Spatial, Object, Goal, Long 四个任务套件）
  • LeRobot 开源真实世界机器人平台（涵盖 126 个不同任务）
  • 模型总参数量约 8.02B，其中可训练参数为 482.34M（占比约 6.01%），基于 OpenVLA 骨干与 LoRA（Rank 32）微调
  • 训练资源：2 块 NVIDIA RTX Pro 6000 GPUs，训练耗时约 5 天；真实世界评估使用 1 块 RTX 5090 GPU
Card 06 评估与结果

评估与结果

  • 评估环境：LIBERO 仿真基准与 LeRobot 真实世界物理平台
  • 主要评估指标：成功率、吞吐量、恢复率、线性探测误差（MAE）
  • 关键实验结果：
  • 在 LIBERO 基准上平均成功率达到 98.1%，在最具挑战性的 LIBERO-Long 上达到 95.3%，显著优于 OpenVLA-OFT 等基线。
  • 在 LeRobot 真实世界长时序推理任务中，性能优势随着任务复杂度增加而扩大，吞吐量保持高水平。
  • 消融实验证明：移除双尺度记忆枢纽导致成功率下降 2.1%，移除 MI 损失下降 0.8%，移除推理令牌下降 1.4%。
  • 错误恢复率达到 12.1%，远高于 OpenVLA 的 5.2% 和无推理令牌变体的 8.7%。
